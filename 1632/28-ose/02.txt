Mówćie bráćiom wáƺym / o ludu mój! y śioſtrom wáƺym / o ty / coś miłośierdźiá doſtąpiłá!
Rozpierájćie śię z mátką wáƺą / rozpierájćie śię ; bo oná nie jeſt żoną moją / á Jam też nie jeſt mężem jey / póki nie odejmie wƺetecżeńſtw ſwojich od oblicża ſwego / á cudzołóſtw ſwych z pośród pierśi ſwojich ;
Bym jey ſnadź nágo nie zewlekł á nie poſtáwił jey / jáką byłá w dźień národzeniá ſwego / y nie ucżynił jey jáko puſtyniá / y nie zoſtáwił jey jáko źiemiá ſuchá / y nie umorzył jey prágnieniem :
Y nád ſynámi jey nie zmiłowáłbym śię / przeto / że ſą ſynámi z wƺetecżeńſtwá /
Bo wƺetecżeńſtwo płodźi mátká ich / ſproſność cżyni rodźićielká ich ; bo mówiłá : Pójdę zá miłośnikámi moimi / którzy mi dodáwáją chlebá mego / y wody mojey / wełny mojey / y lnu mego / oliwy mojey y nápojów mojich.
Przetoż oto Ja zágrodzę ćierniem drogę jey / á ugrodzę płot / áby śćieżek ſwojich nie ználázłá.
Tedy biegáć będźie zá miłośnikámi ſwymi / wƺákże ich nie dogoni ; y ƺukáć ich będźie / ále nie znájdźie. Tedy rzecże : Pójdę á wrócę śię do mężá ſwego pierwƺego ; bo mi lepiej było ná on cżás / niżeli teraz.
Bo oná nie wie tego / żem Ja dáwał jey zboże / y moƺcż / y oliwę ; náwet dáwáłem jey obfitość ſrebrá y złotá / które oni obrácáją ná Báálá.
Dla tego wrócę śię / y zábiorę zboże moje cżáſu jego / y moƺcż mój cżáſu jego / y odbiorę jey wełnę moję / y len mój dány ná okrywánie nágośći jey.
A teraz odkryję ſproſność jey przed ocżymá miłośników jey / á nikt jey nie wyrwie z ręki mojey.
Y ucżynię koniec wƺyſtkiemu weſelu jey / świętom jey / nowym mieśiącom jey / y ſábátom jey / y wƺyſtkim urocżyſtym świętom jey.
Popuſtoƺę też winne máćice jey y figowe drzewá jey / przeto / że mówi : Mojá to zápłátá / którą mi dáli miłośnicy moi ; y obrócę je w láſy / á pożrą je zwierzętá polne.
Y náwiedzę ná niej dni Báálowe / w które im kádźi / á ſtrojąc śię w náuƺnice ſwoje y w klejnoty ſwoje / chodźi zá miłośnikámi ſwymi / ále mnie zápominá / mówi Pán.
Przetoż oto Ja nią łudźić będę / gdy ją wywiodę ná puƺcżę / á łáſkáwie z nią mówić będę ;
Y dam jey winnice jey od tegoż miejſcá / y dolinę Achor / miáſto drzwi nádźiei ; y będźie tám śpiewáłá / jáko zá dni młodośći ſwojey / to jeſt / jáko w dźień / którego wychodźiłá z źiemi Egipſkiej.
A dniá onego mówi Pán / będźieƺ mię zwáłá : Mężu mój! á nie będźieƺ mię więcey zwáłá : Báálu mój!
Bo odejmę imioná Báálów od uſt twojich / że áni wſpomináni będą więcey imieniem ſwojim.
Y ucżynię dla ćiebie przymierze dniá onego z zwierzem polnym / y z ptáſtwem niebieſkiem y z płázem źiemſkim / á łuk y miecż pokruƺę / y wojnę z źiemi znioſę / y ſpráwię / że bezpiecżnie mieƺkáć będą.
Y poślubię ćię ſobie ná wieki : poślubię ćię / mówię / ſobie w ſpráwiedliwośći y w ſądźie y w miłośierdźiu y w litośćiách ;
Poślubię ćię też ſobie w wierze / y poznáƺ Páná.
Dniá onego wyſłuchám / mówi Pán / wyſłuchám / mówi / niebioſá / á one wyſłucháją źiemię ;
A źiemiá wyſłuchá zboże y moƺcż y oliwę / á te rzecży wyſłucháją Jezreelá.
Bo ją ſobie rozśieję ná źiemi / á zmiłuję śię nád tą / co byłá w niełáſce / y rzekę do tego / co nie był ludem moim : Tyś jeſt lud mój! á on rzecże : Tyś jeſt Bóg mój.
