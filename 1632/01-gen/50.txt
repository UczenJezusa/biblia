Zátym upadł Józef ná twarz Ojcá ſwego / y płákał nád nim / á cáłował go.
Y rozkazał Józef ſługom ſwym lekárzom / áby wonnymi máśćiámi námázáli Ojcá jego / y námázáli wonnymi máśćiámi lekárze Izráelá.
A gdy śię <i>mázánia</i> jego wypełniło czterdźieśći dni / ( bo śię ták wypełniáją dni tych którzy wonnymi máśćiámi mázáni bywáją ) tedy go płákáli Egipczánie / przez śiedmdźieśiąt dni.
A po wyjśćiu dni żałoby jego / rzekł Józef do ſług Fáráonowych / mówiąc : Jeſlim teraz nálazł łáſkę w oczách wáƺych / powiedzćie proƺę Fáráonowi mówiąc :
Oćiec mój poprzyśiągł mię mówiąc : Oto ja umieram : w grobie mojim / którym ſobie wykopał w źiemi Chánánejſkiey / tám mię pogrzebieƺ : á teraz niech jádę / proƺę / y pogrzebię Ojcá mego / y záś śię wrócę.
Tedy rzekł Fáráo : Jedź / á pogrzeb Ojcá twego jáko ćię poprzyśiągł.
Jáchał tedy Józef áby pogrzebł Ojcá ſwego : jácháli też z nim wƺyſcy ſłudzy Fáráonowi / <i>tákże</i> ſtárƺy domu jego / y wƺyſcy ſtárƺy źiemie Egipſkiey :
Y wƺyſtek dom Józefów / y bráćia jego / y dom Ojcá jego : tylko dźiatki ſwoje / y owce ſwoje / y woły ſwoje zoſtáwili w źiemi Geſen.
Szły też z nim y wozy / y jezdni : á był poczet bárzo wielki.
Y przyjácháli áż ná pole Atád / które jeſt przy brodźie Jordańſkim / y płákáli tám płáczem wielkim / y bárzo ćiężkim / y obchodźił <i>Józef</i> po Ojcu ſwym żałobę przez śiedm dni.
A ujrzawƺy obywátele źiemie Chánánejſkiey żałobę onę ná polu Atád / mówili : żałobá to ćiężka Egipczánów / przetoż názwano imię <i>miejſcá</i> onego / Abel Mitzrjam / które jeſt przy brodźie Jordańſkim.
Uczynili tedy z nim Synowie jego jáko im był rozkazał.
Y záwieſli go ſynowie jego do źiemie Chánánejſkiey / y pogrzebli go w jáſkini ná polu Máchpelá / którą Abráhám był kupił z rolą / ná ośiádłość grobu od Efroná Hettejczyká / przećiwko Mámre.
Zátym śię wróćił Józef do Egiptu z bráćią ſwą / y ze wƺyſtkimi którzy jeźdźili z nim ná pogrzeb Ojcá jego / odpráwiwƺy pogrzeb Ojcá ſwego.
A widząc bráćia Józefowi że umárł Oćiec ich / mówili : podobno będźie nas miał w nienawiśći Józef / y ſowićie odda nam wƺyſtko złe / któreſmy mu uczynili.
Wſkazáli tedy do Józefá mówiąc : Oćiec twój rozkazał pierwey niż umárł / mówiąc :
Ták powiedzćie Józefowi : Proƺę odpuść teraz przeſtępſtwo bráći twey / y grzech ich / żeć złość wyrządźili : proƺę odpuść teraz wyſtępek ſługom Bogá / Ojcá twego. Y płákał Józef / gdy to mówili do niego.
Y przyſtąpili bráćia jego / á upadƺy przed nim mówili : Otoſmy ſługámi twojimi.
Y rzekł do nich Józef : Nie bójćie śię : bo ázażem ja <i>wam</i> zá Bogá?
Wyśćie złe myślili przećiwko mnie / <i>ále</i> Bóg obróćił to w dobre / chcąc uczynić to / co śię dźiś dźieje / áby záchował ták wielki lud.
A przetoż niebójćie śię / ja żywić będę was y dźiatki wáƺe ; á <i>ták</i> ćieƺył je / y mówił z nimi łágodnie.
Y mieƺkał Józef w Egipćie / ſam / y dom Ojcá jego / á żył Józef ſto y dźieśięć lat.
Y oglądał Józef Syny Efráimowe / áż do trzećiego pokolenia. Synowie też Máchyrá ſyná Mánáſſeſſowego / porodźili śię ná koleniech Józefowych.
Y rzekł Józef do bráćij ſwey : Ja umrę / ále Bóg zápewne náwiedźi was / y wyprowádźi was z źiemie tey do źiemie o którą przyśiągł Abráhámowi / Izáákowi / y Jákóbowi.
Y poprzyśiągł Józef Syny Izráelowe mówiąc : Gdy was náwiedźi Pán Bóg / wynieśćie też kośći moje z tąd.
Y umárł Józef májąc ſto y dźieśięć lat / którego námázawƺy wonnymi máśćiámi / włożono do trumny w Egipćie.
