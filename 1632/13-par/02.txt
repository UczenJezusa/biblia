Cić ſą ſynowie Izráelowi : Ruben / Symeon / Lewi / y Judá / Iſáſchár y Zábulon /
Dán / Józef / y Benjámin / Neftáli / Gád y Aſer.
Synowie Judy : Her / y Onán / y Selá. Ci trzej urodźili mu śię z córki Sui Chánánejſkiej. Ale Her / pierworodny Judy / był złym przed ocżymá Páńſkimi ; przetoż go zábił.
Támár záśię / niewiáſtká jego / urodźiłá mu Fáreſá y Zárę. Wƺyſtkich ſynów Judowych pięć.
Synowie Fáreſowi : Heſron y Hámuel.
Synowie záś Záry : Zámry / y Etán / y Hemán / y Chálkol / y Dárdá ; wƺyſtkich tych było pięć.
A ſynowie Zámrego : Chármi / wnuk Achárowy / który zámięƺánie ucżynił w Izráelu / zgrzeƺywƺy krádźieżą rzecży przeklętych.
A ſynowie Etánowi : Azáryjáƺ.
A ſynowie Eſronowi / którzy mu śię urodźili : Jerámeel / y Rám / y Chálubáj.
Ale Rám ſpłodźił Aminádábá / á Aminádáb ſpłodźił Nááſoná / kśiążęćiá ſynów Judzkich.
A Nááſon ſpłodźił Sálmoná / á Sálmon ſpłodźił Boozá.
A Booz ſpłodźił Obedá / á Obed ſpłodźił Iſájego.
A Iſáj ſpłodźił pierworodnego ſwego Elijábá / y Abinádábá wtórego / y Sámmá trzećiego.
Nátánáelá cżwártego / Ráddájá piątego.
Ozemá ƺóſtego / Dawidá śiódmego.
A śioſtry ich : Sárwijá / y Abigáil ; á ſynowie Sárwii : Abiſáj / y Joáb / y Aſáel / trzej.
A Abigáil urodźiłá Amázę / á oćiec Amázy był Jeter Iſmáelcżyk.
A Káleb / ſyn Heſronowy / ſpłodźił z Azubą / máłżonką ſwoją / y z Jeryjotą ſynów. A ći byli ſynowie jego : Jeſer / y Sobáb / y Ardon.
A gdy umárłá Azubá / pojął ſobie Káleb Eſrátę / która mu urodźiłá Hurá.
A Hur ſpłodźił Ury / á Ury ſpłodźił Beſáleelá.
Potem wƺedł Heſron do córki Máchyrá / ojcá Gáláádowego / á pojął ją / będąc w ƺeśćdźieśiąt lat ; która mu urodźiłá Segubá.
A Segub ſpłodźił Jáirá / który miał dwádźieśćiá trzy miaſt w źiemi Gáláádſkiej.
Bo wźiął Gieſſurytom / y Aſſyryjcżykom wśi Jáirowe / y Kánát z miáſtecżkámi jego / ƺeśćdźieśiąt miaſt. To wƺyſtko pobráli ſynowie Máchyrá / ojcá Gáláádowego.
A gdy umárł Heſron w Káleb Efrátá / tedy żoná Heſronowá Abijá porodźiłá mu Aſſurá / ojcá Tekui.
Byli też ſynowie Jerámeelowi / pierworodnego Heſronowego : Pierworodny Rám / po nim Buná / y Oren / y Oſem z Abii.
Miał tákże drugą żonę Jerámeel / imieniem Atárá ; tá jeſt mátká Onámowá.
Ale ſynowie Rámowi / pierworodnego Jerámeelowego / byli : Máás / y Jámin / y Achár.
Byli też ſynowie Onámowi : Semáj / y Jádá / á ſynowie Semejego : Nádáb y Abiſur ;
A imię żony Abiſurowej Abiháil / która mu urodźiłá Achobbáná y Molidá.
Synowie Nádábowi : Sáled y Affáim ; lecż Sáled umárł bez potomſtwá.
A ſynowie Affáimowi Jeſy ; á ſynowie Jeſy Seſán / á córká Seſáná Achiáláj.
A ſynowie Jády / brátá Semejego / Jeter y Jonátán ; ále Jeter umárł bez potomſtwá.
A ſynowie Jonátánowi : Fálet y Zyzá. Cić byli ſynowie Jerámeelowi.
Lecż nie miał Seſán ſynów / jedno córki ; miał też Seſán ſługę Egipcżániná / imieniem Jeráchá.
Y dał Seſán córkę Jeráchowi / ſłudze ſwemu / zá żonę / która mu urodźiłá Etájá.
Etáj ſpłodźił Nátáná / á Nátán ſpłodźił Zábádá.
A Zábád ſpłodźił Efijálá / á Efijál ſpłodźił Obedá.
A Obed ſpłodźił Jehu / á Jehu ſpłodźił Azáryjáƺá.
A Azáryjáƺ ſpłodźił Heleſá / á Heles ſpłodźił Eláſá.
A Elás ſpłodźił Syſmájá / á Syſmáj ſpłodźił Sellumá.
A Sellum ſpłodźił Ikámijáƺá / á Ikámijáƺ ſpłodźił Eliſámá.
Y ſynowie Kálebá / brátá Jerámeelowego : Meſá pierworodny jego / który był ojcem Zyfejcżyków y ſynów Máreſy / ojcá Hebronowego.
A ſynowie Hebronowi : Kore y Táfuá / y Rechem / y Semmá.
A Semmá ſpłodźił Ráhámá / ſyná Jerkáámowego / á Rechem ſpłodźił Sámmájego.
A Sámmáj był ſynem Máonowym / á Máon był ojcem Betſurcżyków.
Efá też / záłożnicá Kálebowá / urodźiłá Háráná / y Moze / y Giezezá ; á Hárán ſpłodźił Giezezá.
A ſynowie Jáchdájowi : Regiem / y Jotám / y Gieſán / y Fálet / y Efá / y Sááf /
Záłożnicá záś drugá Kálebowá Mááchá urodźiłá Sáberá / y Tyrcháná.
A żoná Sááfowá urodźiłá ojcá Mádmeńcżyków / y Sewá / ojcá Máchbeńcżyków / y ojcá Gábáońcżyków ; á córká Kálebowá byłá Achſá.
Cić byli ſynowie Kálebá / ſyná Hurowego / pierworodnego Efráty : Sobál / oćiec Káryjátyjárymcżyków.
Sálmá / oćiec Betlehemcżyków / Cháref / oćiec Betgádercżyków.
Miał też ſynów Sobál / oćiec Káryjátyjárymcżyków / który doglądał połowy Menuchoty.
A domy Káryjátyjárymſkie były Jetrejcżycy / y Futejcżycy / y Semátejcżycy / y Máſerejcżycy / z których też poƺli Sáráitowie / y Eſtáolitowie.
A ſynowie Sálmy : Betlehemcżycy / y Netofátcżycy / ozdoby domu Joábowego / y połowá Mánácháty / ojcá Sorygo.
A domy piſárzów mieƺkájących w Jebes : Tyryjátejcżycy / Symátejcżycy / Suchátejcżycy. Cić ſą Cynejcżycy / którzy poƺli z Hemátá / ojcá domu Rechábowego.
