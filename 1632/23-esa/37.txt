A gdy to uſłyƺał król Ezechyjáƺ / rozdárł ƺáty ſwoje / á oblókƺy śię w wór / wƺedł do domu Páńſkiego.
Y poſłał Elijákimá / ſpráwcę domu ſwego / y Sobnę piſárzá / y ſtárƺych z kápłanów oblecżonych w wory / do Izájáƺá proroká / ſyná Amoſowego.
Którzy rzekli do niego : Ták mówi Ezechyjáƺ : Dźień ten jeſt dźień utrapienia / y łájánia / y bluźnieniá ; ábowiem ſynowie przyƺli áż do porodzeniá / ále śiły niemáƺ ku rodzeniu.
Oby uſłyƺał Pán / Bóg twój / ſłowá Rábſáceſowe / którego poſłał król Aſſyryjſki / pán jego / áby urągał Bogu żyjącemu / y pomśćił śię Pán Bóg twój / tych ſłów / które ſłyƺał! Przetoż ucżyń modlitwę zá te oſtátki ludu / które śię znájdują.
Przyƺły tedy ſłudzy królá Ezechyjáƺá do Izájáƺá ;
Którym odpowiedźiał Izájáƺ : Ták powiedzćie Pánu wáƺemu / ták mówi Pán : Nie bój śię tych ſłów / któreś ſłyƺał / któremi mię lżyli ſłudzy królá Aſſyryjſkiego.
Oto ja mu dam innego duchá / áby uſłyƺáwƺy wieść náwróćił śię do źiemi ſwojey ; y ſpráwię to / że polegnie od miecżá w źiemi ſwojey.
Ale Rábſáces wróćiwƺy śię ználázł królá Aſſyryjſkiego dobywájącego Lebny ; ábowiem uſłyƺał / iż odćiągnął był od Láchys.
A uſłyƺáwƺy o Tyráku / królu Etyjopſkim / że mówiono : Oto ćiągnie / áby wálcżył przećiwko tobie ; uſłyƺáwƺy to / mówię / przećie poſłał poſłów do Ezechyjáƺá z temi ſłowy :
To powiedzćie Ezechyjáƺowi / królowi Judzkiemu / mówiąc : Niech ćię nie zwodźi Bóg twój / któremu ty ufáƺ / á mówiƺ : Nie będźie podáne Jeruzálem w ręce królá Aſſyryjſkiego.
Otoś ſłyƺał / co pocżynili królowie Aſſyryjſcy wƺyſtkim źiemiom / które wygłádźili ; á tybyś miał być wybáwiony?
Izali je wybáwili bogowie tych narodów / które wygubili ojcowie moi : Gozán / y Hárán / y Reſef / y ſynów Eden / którzy byli w Teláſſár?
Gdźież jeſt król Elmát / y król Arfád / y król miáſtá Sefárwáim / Aná / y Awá?
Przetoż wźiąwƺy Ezechyjáƺ liſt z ręki poſłów / przecżytał go / á wƺedƺy do domu Páńſkiego / rozćiągnął go Ezechyjáƺ przed Pánem.
Y modlił śię Ezechyjáƺ Pánu / mówiąc :
Pánie zaſtępów / Boże Izráelſki / śiedzący ná Cherubinách! Ty / tyś ſám jeſt Bóg wƺyſtkich króleſtw źiemi / tyś ſtworzył niebo y źiemię.
Nákłońże / Pánie! uchá twego / á uſłyƺ ; otwórz / Pánie! ocży twoje / á obácż ; uſłyƺ wƺyſtkie ſłowá Sennácherybowe / który przyſłał háńbić ćiebie / Bogá żywego.
Práwdáć jeſt / Pánie! że ſpuſtoƺyli królowie Aſſyryjſcy wƺyſtkie te kráiny / y źiemię ich ;
Y powrzucáli bogów ich w ogień ; ábowiem nie byli bogámi / ále robotą rąk ludzkich / drewno y kámień ; przetoż ich wygubili.
A teraz / o Pánie / Boże náƺ! wybáw nas z ręki jego / áby poznáły wƺyſtkie króleſtwá źiemi / żeś ty / Pánie! ſám Bogiem.
Tedy poſłał Izájáƺ / ſyn Amoſowy / do Ezechyjáƺá / mówiąc : Ták mówi Pán Bóg Izráelſki : O coś mię prośił z ſtrony Sennácherybá / królá Aſſyryjſkiego /
Tedy to jeſt ſłowo / które mówił Pán o nim : Pánná / córká Syońſká / wzgárdźiłá ćię / śmiáłá śię z ćiebie / kiwáłá głową zá tobą córká Jeruzálemſká.
Kogożeś háńbił / y kogoś bluźnił? przećiwko komużeś podnióſł głos / y wynioſłeś ku górze ocży ſwe? przećiwko Świętemu Izráelſkiemu.
Przez ſługi twoje háńbiłeś Páná / y mówiłeś : W mnóſtwie wozów mojich wſtąpiłem ja ná wyſokie góry / ná ſtrony Libáńſkie / y porąbię wyſokie cedry jego / y wyborne jodły jego ; y wnijdę ná ſámę wyſokość wierzchu jego / do láſów / y urodzájnych ról jego.
Jam wykopał źródłá y piłem wody / á wyſuƺyłem ſtopámi nóg mojich wƺyſtkie potoki miejſc oblężonych.
Izáżeś nie ſłyƺał / żem to z dáwná ucżynił / y ode dni ſtárodáwnych to ſpráwił? A teraz do tego przywodzę / áby w puſtynie y w kupy rumu miáſtá obronne obrócone były.
A obywátele ich ręce ſkurcżone májąc / przeſtráƺeni ſą y záwſtydzeni / ſtáli śię jáko tráwá polná / y jáko źiele wſchodzące / y tráwy ná dáchách / á śiewy rdzą zepſowáne / pierwej niżeliby doroſły.
Mieƺkánie twoje / y wyjśćie twoje / y wejśćie twoje znam / y popędliwość twoję przećiwko ſobie.
Poniewáżeś śię przećiwko mnie zájuƺył / á zápędy twoje przyƺły do uƺu mojich / przetoż záłożę kolce moje zá nozdrzá twoje / á wędźidło moje wpráwię w gębę twoję / y wrócę ćię tą drogą / którąś przyƺedł.
A to miej zá znák / Ezechyjáƺu! Tego roku jeść będźieƺ ſámorodne zboże / tákże y drugiego roku ſámorodne zboże ; ále roku trzećiego będźiećie śiáć y żąć / y winnice ſádźić / y pożywáć owoce ich.
Oſtátek bowiem domu Judy / który pozoſtał / wkorzeni śię głęboko / y wydá owoc ku górze.
Abowiem z Jeruzálemu wyjdą oſtátki / y záchowáni z góry Syońſkiej. Gorliwość Páná zaſtępów to ucżyni.
Przetoż ták mówi Pán o królu Aſſyryjſkim : Nie wnijdźie do miáſtá tego / áni tám ſtrzáły wyſtrzeli / áni go záprzątnie tárcżá / áni uſypie około niego ƺáńców.
Drogą / którą przyƺedł / záś śię wróći / á do miáſtá tego nie wnijdźie / mówi Pán.
Bo będę bronił miáſtá tego / y záchowám je ſám dla śiebie / y dla Dawidá / ſługi mego.
Tedy wyƺedł Anjoł Páńſki / y pobił w oboźie Aſſyryjſkim ſto ośmdźieśiąt / y pięć tyśięcy ; á gdy wſtáli bárzo ráno / oto wƺędy pełno trupów.
Przetoż ruƺywƺy śię / odjechał / y wróćił śię Sennácheryb / król Aſſyryjſki / á mieƺkał w Niniwie.
A gdy chwálił Neſrochá / bogá ſwego / w domu / tedy Adrámelech y Sáráſſár / ſynowie jego / zábili go miecżem / á ſámi ućiekli do źiemi Arárát ; á królował Aſſárháddon / ſyn jego / miáſto niego.
