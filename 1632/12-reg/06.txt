Y rzekli Synowie Proroccy do Elizeuƺá : Oto miejſce ná którym mieƺkamy przed tobą / ćiáſne jeſt dla nas :
Niech idźiemy / proƺę áż do Jordanu / á weźmiemy z támtąd káżdy po jednym drzewie / y zbudujemy tám ſobie miejſce ku mieƺkániu. Którym on rzekł : Idźćie.
Y rzekł jeden z nich : Pódź proƺę y <i>ty</i> z ſługámi twojimi : A on rzekł : Y ja pójdę : Y ƺedł z nimi.
A przyƺedƺy do Jordanu / rąbáli drzewo.
Y ſtáło śię / gdy jeden z nich obalał drzewo / że <i>mu</i> śiekierá wpádłá w wodę : y záwołał / mówiąc : Ach ách Pánie mój! y táć <i>byłá</i> pożycżána.
Rzekł tedy mąż Boży ; Gdźieƺ upádłá / y ukazał mu miejſce : á on ućiąwƺy drewno / wrzućił tám / y ſpráwił że wſpłynęłá oná śiekierá.
Y rzekł ; Weźmi <i>ją</i> ſobie : który zćiągnąwƺy rękę ſwą / wźiął ją.
A gdy Król Syryjſki walcżył z Izráelem / y nárádzał śię z ſługámi ſwojimi / mówiąc : Ná tym á ná tym miejſcu położy śię wojſko moje.
Tedy poſłał mąż Boży do Królá Izráelſkiego / mówiąc : Strzeż śię ábyś nie przechodźił przez ono miejſce / bo tám Syryjcżycy ſą ná záſadzce.
Przetoż poſłał Król Izráelſki ná ono miejſce / o którym mu był powiedźiał mąż Boży / y przeſtrzegł go áby śię go chronił / nie raz áni dwá.
A ták zátrwożyło śię ſerce Królá Syryjſkiego dlá tego : Przetoż zwoławƺy ſług ſwojich / rzekł do nich ; Cżemuż mi nie powiećie kto wżdy z nas <i>donośi to</i> Królowi Izráelſkiemu?
Y rzekł jeden z ſług jego : Nie ták Królu Pánie mój : ále Elizeuƺ Prorok / który jeſt w Izráelu / oznájmuje Królowi Izráelſkiemu ſłowá które mówiƺ w tájemnym pokoju twojim.
A <i>on</i> rzekł : Idźćie / á dowiedzćie śię gdźie jeſt / ábym poſłał y pojimał go : y powiedźiano mu / mówiąc : Oto jeſt w Dotánie.
Przetoż / poſłał tám konie y wozy z wielkim wojſkiem / którzy przyćiągnąwƺy w nocy oblegli miáſto.
Tedy wſtawƺy ráno ſługá mężá Bożego / wyƺedł / á oto wojſko obtocżyło miáſto / y konie / y wozy. Y rzekł ſługá jego do niego : Ach Pánie mój / cóż mamy cżynić?
A on odpowiedźiał : Nie bój śię : bo więcey ich z námi niż z nimi.
Modlił śię tedy Elizeuƺ / y rzekł : O PANIE otwórz / proƺę / ocży jego / żeby widźiał : Y otworzył PAN ocży ſługi onego / y ujrzał / á oto górá pełna koni / y wozy ogniſte około Elizeuƺá.
A gdy <i>Syryjcżycy</i> ƺli do niego / modlił śię Elizeuƺ Pánu / mówiąc : Proƺę záraź ten lud ślepotą : y záráźił je PAN ślepotą według ſłowá Elizeuƺowego.
W tym rzekł do nich Elizeuƺ : Nie táć to drogá / áni to miáſto : Pódźćie zá mną / á záwiodę was do mężá którego ƺukaćie. Y przywiódł je do Sámáryjey.
A gdy weƺli do Sámáryjey / rzekł Elizeuƺ : O PAnie / otwórz ocży tych / áby przejrzeli. Y otworzył PAN ocży ich / y widźieli że <i>byli</i> w pośrzód Sámáryjey.
Y rzekł Król Izráelſki do Elizeuƺá gdy je ujrzał :
Mamże je pobić Ojcże mój? Ale <i>on</i> rzekł : Nie bij. Azażeś je wźiął przez miecż twój / álbo przez łuk twój / żebyś je miał pobić? Połóż chleb y wodę przed nie / áby jedli y pili / y wróćili śię do Páná ſwego.
A ták przygotował dla nich doſtátek wielki y jedli y pili / y puśćił je / y odeƺli do Páná ſwego. Y nie ważyły śię więcey wojſká Syryjſkie wpadáć do źiemie Izráelſkiey.
Stáło śię potym / że zebrał Benádád Król Syryjſki wƺyſtkie wojſká ſwe / á przyćiągnął y obległ Sámáryją.
Przetoż był głód wielki w Sámáryjey / ábowiem ją było oblężono / ták / iż głowę oślą <i>przedawano</i> zá ośmdźieśiąt śrebrników : á cżwartą cżęść miáry gnoju gołębiego / zá pięć śrebrników.
Y przydáło śię / gdy Król Izráelſki przechadzał śię po murze / że <i>jedná</i> niewiáſtá záwołáłá nań / mówiąc : Rátuj mię Królu Pánie mój!
Który rzekł : Nie rátujeli ćię PAN / z kądże <i>ja</i> ćiebie porátuję? izali z gumná / cżyli z práſy?
Nád to rzekł jey Król : Cóż ći? A oná rzekłá : Tá niewiáſtá rzekłá do mnie ; Daj Syná twego / żebyſmy go zjádły dźiśia / á jutro zjemy Syná mego.
Y uwárzyłyſmy Syná mego / y zjádłyſmy go : Potym rzekłám jey dniá drugiego. Daj Syná twego ábyſmy go zjádły : ále oná ſkryłá Syná ſwego.
A gdy Król uſłyƺał ſłowá oney niewiáſty / rozdárł odźienie ſwe / ( á gdy śię przechodźił po murze / widźiał lud że wór był ná ćiele jego od ſpodku. )
Tedy rzekł <i>Król</i> : To niechaj mi ucżyni Bóg / y to przycżyni / jeſli śię głowá Elizeuƺá / Syná Sáfátowego ná nim dźiśia oſtoji.
( A Elizeuƺ śiedźiał w domu ſwojim / y Stárƺy śiedźieli z nim. ) Y poſłał mężá z tych którzy przed nim ſtali : á pierwey niż on poſeł przyƺedł do niego / rzekł był do Stárƺych : Nie wiećieƺ iż poſłał Syn tego mężobójcę áby śćięto głowę moję? Pátrzćież gdy przydźie ten poſeł / zámknićieƺ drzwi / á záhámujćie go przede drzwiámi : Boć tęntęn nóg Páná jego jeſt już zá nim.
A gdy to jeƺcże mówił z nimi / oto poſeł przychodźił ku niemu / y rzekł : Oto to złe jeſt od PANá / cżegoż mam więcey ocżekawáć od PAná.
