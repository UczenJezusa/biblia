Y ja bráćia nie mogłem wam mówić jáko duchownym / ále jáko ćieleſnym / <i>y</i> jáko niemowiątkom w CHryſtuśie.
Nápawałem was mlekiem / á nie <i>karmiłem</i> was pokármem / bośćie jeƺcże nie mogli <i>znieść</i> ; owƺem y teraz jeƺcże nie możećie :
Gdyż jeƺcże ćieleſnymi jeſteśćie. Bo ponieważ miedzy wámi <i>jeſt</i> zazdrość / y ſwary / y roſterki / ázażeśćie nie ćieleśni / y według cżłowieká nie chodźićie?
Abowiem gdy kto mówi : Jam jeſt Páwłów ; á drugi / jam Apolloſów ; ázaż ćieleſnymi nie jeſteśćie?
Bo któż jeſt Páweł? Kto Apollos? Jedno ſłudzy przez któreśćie uwierzyli ; á to jáko káżdemu PAN dał.
Jam ƺcżepił / Apollos polewał ; ále Bóg wzroſt dał.
A ták áni ten co ƺcżepi jeſt cżym / áni ten co polewa ; ále Bóg który wzroſt dawa.
Lecż ten który ƺcżepi / y ten który polewa / jedno ſą : á káżdy ſwoję zapłátę weźmie według prace ſwojey.
Abowiem jeſteſmy pomocnikámi Bożymi : <i>Wy</i> Bożą rolą / Bożym budynkiem jeſteśćie.
Według łáſki Bożey / która mi jeſt dána jáko mądry budownicży záłożyłem grunt / á drugi ná nim buduje : wƺákże káżdy niechaj bacży jáko ná nim buduje.
Abowiem gruntu innego nikt nie może záłożyć / oprócż tego który jeſt záłożony / który jeſt JEzus CHryſtus.
A jeſli kto ná tym grunćie buduje / złoto / śrebro / kámienie drogie / drwá / śiáno / ſłomę ;
Káżdego robotá jáwna będźie : bo to dźień pokaże. Gdyż przez ogień objáwiona będźie ; á káżdego roboty jáká jeſt / ogień doświadcży.
Jeſli cżyjá robotá zoſtánie którą ná nim budował / zapłátę weźmie.
Jeſli cżyjá robotá zgore / <i>ten</i> ƺkodę podejmie : lecż on ſam będźie záchowány ; wƺákże ták jáko przez ogień.
Azaż nie wiećie / iż kośćiołem Bożym jeſteśćie / á Duch Boży mieƺka w was?
<i>A</i> jeſli kto gwałći kośćiół Boży / tego Bóg ſkáźi : ábowiem kośćiół Boży święty jeſt / którym wy jeſteśćie.
Niechajże nikt ſámego śiebie nie zwodźi. Jeſli śię kto <i>ſobie</i> zda być mądrym miedzy wámi ná tym świećie / niech śię ſtánie głupim áby śię ſtał mądrym.
Abowiem mądrość tego świátá / głupſtwem jeſt u Bogá. Bo nápiſano ; Który chwyta mądre w chytrośći ich.
Y záśię : PAN zna myśli mądrych / iż ſą márnośćią.
A ták niech śię nikt nie chlubi ludźmi : ábowiem wƺyſtkie rzecży ſą wáƺe.
Bądź Páweł / bądź Apollos / bądź Cefás / bądź świát / bądź żywot / bądź śmierć / bądź przytomne / bądź przyƺłe <i>rzecży</i> ; wƺyſtkie <i>ſą</i> wáƺe.
Aleśćie wy Chryſtuſowi / á CHryſtus Boży.
