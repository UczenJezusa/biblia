Y ſtáło śię roku śiódmego / mieśiącá piątego / dźieśiątego dniá tegoż mieśiącá / przyƺli niektórzy z ſtárƺych Izráelſkich / áby śię rádźili Páná ; y uśiedli przedemną.
Tedy śię ſtáło ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! mów do ſtárƺych Izráelskich / á powiedz im : Ták mówi pánujący Pán : Izali wy przychodźićie / ábyśćie śię mnie rádźili? Jáko żyję Ja / że śię wy mnie nie rádźićie / mówi pánujący Pán ;
Izali śię zá nich záſtáwiáć będźieƺ? Izali śię zá nich záſtáwiáć będźieƺ / ſynu cżłowiecży? Oznájmij im rácżey obrzydliwośći ojców ich /
A rzecż do nich : Ták mówi pánujący Pán : Tego dniá / któregom wybrał Izráelá / podnioſłem rękę moję náśieniu domu Jákóbowego / y dáłem śię im poznáć w źiemi Egipſkiej ; podnioſłem rękę moję dla nich / mówiąc : Jam Pán / Bóg wáƺ.
Onego dniá podnioſłem rękę moję dla nich / że ich wywiodę z źiemi Egipſkiej do źiemi / którąm im upátrzył / opływájącey mlekiem y miodem / która jeſt ozdobą wƺyſtkich źiem.
Y rzekłem im : Káżdy z was niech porzući obrzydliwośći ocżów ſwojich / á nie kálájćie śię plugáwemi báłwánámi egipſkiemi ; bom Ja Pán / Bóg wáƺ.
Ale mi byli odpornymi / y nie chćieli mię ſłucháć ; żáden z nich obrzydliwośći ocżów ſwojich nie odrzućił / y plugáwych báłwánów egipſkich nie opuśćił ; przetożem rzekł : Wyleję gniew mój ná nich / á wypełnię popędliwość moję nád nimi w pośrzodku źiemi egipſkiej.
A wƺákżem ucżynił dla imienia mego / áby nie było zelżone przed ocżymá tych narodów / miedzy którymi oni byli / przed których ocżymá dáłem śię im poznáć / że ich chcę wywieść z źiemi egipſkiej.
A ták wywiodłem ich z źiemi egipſkiej / y przyprowádźiłem ich ná puƺcżę ;
Y dáłem im uſtáwy moje / y ſądy moje podáłem im do wiádomośći ; które jeſliby cżłowiek záchowywał / żyć w nich będźie.
Nádto y ſábáty moje dáłem im / áby były znákiem miedzy mną y miedzy nimi / áby wiedźieli / iżem Ja jeſt Pán / który ich poświęcám.
Ale mi śię odpornym ſtał dom Izráelſki ná puƺcży ; w uſtáwách mojich nie chodźili / y ſądy moje odrzućili / które jeſliby cżłowiek cżynił / żyć w nich będźie. Sábáty też moje bárzo zgwáłćili ; przetożem mówił / iż wyleję popędliwość moję ná nich ná puƺcży / ábym ich wygubił.
Lecżem ucżynił dla imienia mego / áby nie było zelżone przed ocżymá tych narodów / przed którychem ich ocżymá wywiódł.
A wƺákżem podnióſł rękę moję dla nich ná tey puƺcży / że ich nie wprowádzę do źiemi / którąm im był dał / opływájącey mlekiem y miodem / y która jeſt ozdobą wƺyſtkich źiem ;
Ponieważ ſądy moje odrzućili / á w uſtáwách mojich nie chodźili / y ſábáty moje pogwáłćili / á że ſerce ich zá plugáwemi báłwánámi chodźi.
Ale im przećie przepuśćiło oko moje / ták / żem ich nie wytráćił / y nie wygłádźił do ƺcżętu ná puƺcży.
Y mówiłem do ſynów ich ná tey puƺcży : W uſtáwách ojców wáƺych nie chodźćie / y ſądów ich nie przeſtrzegájćie / áni śię plugáwemi báłwánámi ich kálájćie.
Jam Pán / Bóg wáƺ ; w uſtáwách mojich chodźćie / á ſądów mojich ſtrzeżćie / y cżyńćie ich ;
Sábáty też moje święććie / y będą znákiem miedzy mną y miedzy wámi / ábyśćie wiedźieli / żem Ja Pán / Bóg wáƺ.
Lecż mi byli odpornymi ći ſynowie ; w uſtáwách mojich nie chodźili / y ſądów mojich nie przeſtrzegáli / áby je cżynili / ( które jeſliby cżynił cżłowiek pewnieby żył w nich ) y ſábáty moje pogwáłćili. Y rzekłem : Wyleję popędliwość moję ná nich / ábym wykonał gniew ſwój ná nich ná tey puƺcży.
Alem odwróćił rękę moję / á ucżyniłem to dla imienia mego / áby nie było zelżone przed ocżymá tych narodów / przed którychem ich ocżymá wywiódł.
Alem ja podnióſł rękę moję dla nich ná puƺcży / żem ich miał rozproƺyć miedzy pogán / y rozwiáć ich po źiemiách /
Przeto / że ſądów mojich nie cżynili / y uſtáwy moje odrzućili / y ſábáty moje pogwáłćili / á zá plugáwemi báłwánámi ojców ſwojich ocży ſwe obróćili.
Dlátegoż y Jam im dał uſtáwy nie dobre y ſądy / w których żyć nie będą ;
Y ſplugáwiłem ich z dárámi ich / gdy przewodźili przez ogień wƺelkie otwárzájące żywot / ábym ich ſpuſtoƺył / á żeby śię dowiedźieli / żem ja Pán.
Przetoż mów do domu Izráelſkiego / ſynu cżłowiecży! á powiedz im : Ták mówi pánujący Pán : Jeƺcże y w tym lżyli mię ojcowie wáśi / dopuƺcżájąc śię przećiwko mnie przeſtępſtwá /
Że gdym ich wwiódł do źiemi / o którąm był podnióſł rękę moję / żem ją im dáć miał / gdźie ujrzeli jáki págórek wyſoki / álbo jákie drzewo gáłęźiſte / záráz tám ofiárowáli ofiáry ſwoje / y dáwáli tám dráźniące dáry ſwoje / támże kłádli y wdźięcżną wonność ſwoję / támże ſpráwowáli mokre ofiáry ſwoje.
A choćiáżem mówił do nich : Cóż to zá wyżyná / do którey wy chádzáćie? Przećie ją zowią wyżyną áż do dniá tego.
Przetoż rzecż domowi Izráelſkiemu : Ták mówi pánujący Pán : Izali śię wy drogámi ojców wáƺych káláć máćie / á z obrzydliwośćiámi ich nierząd płodźić?
Y máćież śię káláć przy wƺyſtkich plugáwych báłwánách wáƺych / przynoƺąc dáry wáƺe y przewodząc ſynów wáƺych przez ogień áż do dniá tego / á przećie odemnie rády ƺukáć / o domie Izráelſki? Jáko żyję Ja / mówi pánujący Pán / że śię wy mnie więcey rádźić nie będźiećie.
A to / co wſtępuje ná myśl wáƺę / nigdy śię nie ſtánie. Wy mówićie : Będźiemy jáko inne narody / jáko pokolenia innych źiem / drewnu y kámieniowi ſłużące ;
Jáko żyję Ja / mówi pánujący Pán / że ręką możną y rámieniem wyćiągnionem / á w popędliwośći wyláney będę królował nád wámi ;
Y wywiodę was z narodów / á zgromádzę was z źiem / do którycheśćie rozproƺeni / ręką możną / y rámieniem wyćiągnionem / y w popędliwośći wyláney ;
A prowádząc was po puƺcży tych narodów / tám śię z wámi twárzą w twárz ſądźić będę.
Jákom śię ſądźił z ojcámi wáƺymi ná puƺcży źiemi Egipſkiej / ták śię z wámi ſądźić będę / mówi pánujący Pán ;
Y popędzę was pod rózgą / ábym was przywiódł do związku przymierza.
Ale z was wybiorę odpornych y wyſtępujących przećiwko mnie ; z źiemi pielgrzymſtwá ich wywiodę ich / wƺákże do źiemi Izráelſkiej nie wnijdą ; y poznáćie / żem Ja Pán.
Wy tedy / o domie Izráelſki! ták mówi pánujący Pán : Idźćież / ſłużćie káżdy plugáwym báłwánom ſwoim y ná potem / ponieważ mnie nie ſłucháćie ; á imienia mego świętego nie kálájćie więcey dárámi wáƺemi / y plugáwemi báłwánámi wáƺemi.
Bo ná świętey górze mojey / ná wyſokiej górze Izráelſkiej / mówi pánujący Pán / tám mi ſłużyć będźie wƺyſtek dom Izráelſki / ile ich będźie w oney źiemi ; tám ich łáſkáwie przyjmę / y tám śię pytáć będę o ofiárách wáƺych podnoƺonych / y o pierwiáſtkách dárów wáƺych ze wƺyſtkiemi świętymi rzecżámi wáƺemi.
Z wdźięcżną wonnośćią łáſkáwie was przyjmę / gdy was wywiodę z narodów / á zgromádzę was z onych źiem / do którycheśćie rozproƺeni byli / á ták poświęcony będę w was przed ocżymá onych narodów.
A poznáćie / żem Ja Pán / gdy was wprowádzę do źiemi Izráelſkiej / do źiemi oney / o którąm podnióſł rękę moję / że ją dam ojcom wáƺym ;
A wſpomnićie tám ná drogi wáƺe / y ná wƺyſtkie ſpráwy wáƺe / któremiśćie śię ſplugáwili / á obmierzniećie ſámi ſobie przed oblicżem wáƺem dla wƺyſtkich złośći wáƺych / któreśćie cżynili.
Tám śię dowiećie / żem Ja Pán / gdy wam to ucżynię dla imienia mego / nie według dróg wáƺych złych / áni według ſpráw wáƺych ſkáżonych / o domie Izráelſki! mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! obróć twárz twoję ku ſtronie południowej / á krop jáko roſą ná południe / y prorokuj przećiwko láſowi polá południowego /
A rzecż láſowi południowemu : Słucháj ſłowá Páńſkiego : Ták mówi pánujący Pán : Oto Ja rozniecę w tobie ogień / który pożre w tobie wƺelkie drzewo źielone / y wƺelkie drzewo ſuche ; nie będźie ugáƺony płomień páłájący / y zgoreją w nim wƺyſtkie twárze od południá áż do północy.
Y ujrzy wƺelkie ćiáło / żem go Ja Pán zápálił ; nie będźie ugáƺony.
Y rzekłem : Ach pánujący Pánie! Oni mówią o mnie : Ten tylko w przypowieśćiách mówi.
