Słowo / które śię ſtáło do Jeremijáƺá od Páná roku dźieśiątego Sedekijáƺá / królá Judzkiego / który jeſt rok ośmnáſty Nábuchodonozorá.
( A wtencżás wojſko królá Bábilońſkiego obległo było Jeruzálem / á Jeremijáƺ prorok zámknięty był w śieni ćiemnicy / która byłá w domu królá Judzkiego.
Bo go był dał wſádźić Sedekijáƺ / król Judzki / mówiąc : Cżemu ty prorokujeƺ / mówiąc : Ták mówi Pán : Oto Ja to miáſto podám w rękę królá Bábilońſkiego / y weźmie je?
Sedekijáƺ tákże król Judzki nie ujdźie ręki Cháldejcżyków ; ále zápewne wydány będźie w rękę królá Bábilońſkiego / y będą mówiły uſtá jego z uſty jego / á ocży jego ocży jego oglądáją.
Y záwiedźie Sedekijáƺá do Bábilonu / áby tám był / áż go náwiedzę / mówi Pán ; ponieważ wálcżyćie z Cháldejcżykámi / nie poƺcżęśći śię wam. )
Tedy rzekł Jeremijáƺ : Stáło śię ſłowo Páńſkie do mnie / mówiąc :
Oto Chánámeel / ſyn Sállumá / ſtryjá twego / przydźie do ćiebie / mówiąc : Kup ſobie rolę moję / która jeſt w Anátot ; bo tobie náleży práwem bliſkośći / ábyś ją kupił.
A gdy przyƺedł do mnie Chánámeel / ſyn ſtryjá mego / według ſłowá Páńſkiego do śieni ćiemnicy / y rzekł do mnie : Proƺę / kup rolę moję / która jeſt w Anátot / które jeſt w źiemi Benjáminowej / bo tobie náleży práwem dźiedźicżnem / y bliſkośćiąć przypádá / kupże ją ſobie : tedy zrozumiáwƺy / że to było ſłowo Páńſkie /
Kupiłem od Chánámeelá / ſyná ſtryjá ſwego / onę rolę / która jeſt w Anátot / y odwáżyłem mu pieniędzy / śiedmnaśćie ſyklów ſrebrá ;
A ucżyniwƺy zápis zápiecżętowáłem y oświádcżyłem świádkámi / odwáżywƺy pieniądze ná wádze.
Potemem wźiął według przykázánia y práwá zápis kupná zápiecżętowány y otwárty ;
Y oddáłem on zápis kupná Báruchowi / ſynowi Neryjáƺá / ſyná Mááſejáƺowego / przed ocżymá Chánámeelá / ſtryjá ſwego / y przed ocżymá świádków / którzy śię podpiſáli w onym zápiśie kupná / przed ocżymá wƺyſtkich Żydów / którzy byli uśiedli w śieni ćiemnicy ;
Y rozkazałem Báruchowi przed ocżymá ich / mówiąc :
Ták mówi Pán zaſtępów / Bóg Izráelſki : Weźmij te zápiſy / ten zápis tego kupná / jáko zápiecżętowány / ták y ten zápis otworzony / á włóż je w nacżynie gliniáne / áby trwáły przez wiele lat ;
Bo ták mówi Pán zaſtępów / Bóg Izráelſki : Jeƺcże będą kupowáne domy / y role / y winnice w tey źiemi.
Potem modliłem śię Pánu / kiedym oddał on zápis kupná Báruchowi / ſynowi Neryjáƺowemu / mówiąc :
Ach / pánujący Pánie! otoś ty ucżynił niebo y źiemię mocą ſwoją wielką y rámieniem twoim wyćiągnionem / nie jeſtći ſkrytá przed tobą żádná rzecż ;
Cżyniƺ miłośierdźie nád tyśiącámi / y oddájeƺ niepráwość ojcowſką do łoná ſynów ich po nich ; Bóg wielki / mocny / Pán zaſtępów imię twoje ;
Wielki w rádźie y możny w ſpráwie / ponieważ ocży twoje otworzone ſą ná wƺyſtkie drogi ſynów ludzkich / ábyś oddał káżdemu według dróg jego / y według owoców ſpráw jego ;
Któryś ucżynił znáki y cudá ná źiemi Egipſkiej áż do dniá tego / y w Izráelu / y miedzy innymi ludźmi / y ucżyniłeś ſobie imię / jáko śię to dźiś okázuje.
Boś wywiódł lud twój Izráelſki z źiemi Egipſkiej w znákách y w cudách / y w ręce mocney / y w rámieniu wyćiągnionem y w ſtráchu wielkim ;
A podáłeś im tę źiemię / o którąś przyśiągł ojcom ich / żeś im miał dáć źiemię opływájącą mlekiem y miodem.
Ale że wƺedƺy do niej / á pośiádƺy ją / nie uſłucháli głoſu twojego / y w zakonie twoim nie chodźili / wƺyſtkiego / coś im rozkázał cżynić / nie cżynili : przetoż ſpráwiłeś to / áby nań przyƺło to wƺyſtko złe.
Oto ſtrzelbę zátocżono przećiwko miáſtu / áby je wźięto / á miáſto podáne jeſt w ręce Cháldejcżyków wálcżących przećiw niemu przez miecż / y głód / y mór ; á ták cośkolwiek rzekł / dźieje śię / jáko to ſám widźiƺ.
A ty przećie mówiƺ do mnie / pánujący Pánie : Kup ſobie tę rolę zá pieniądze / á oświádcż to świádkámi / choć już to miáſto podáne jeſt w ręce Cháldejcżyków.
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá / mówiąc :
Otom Ja Pán / Bóg wƺelkiego ćiáłá ; izáliż przedemną może być ſkrytá która rzecż?
Przetoż ták mówi Pán : Oto Ja dáję to miáſto w rękę Cháldejcżyków / y w rękę Nábuchodonozorá / królá Bábilońſkiego / y weźmie je.
A wƺedƺy Cháldejcżycy / którzy wálcżą przećiwko temu miáſtu / zápálą to miáſto ogniem / y ſpálą je y te domy / ná których dáchách kádźili Báálowi / á ſpráwowáli ofiáry mokre bogom cudzym / áby mię wzruƺáli do gniewu.
Bo ſynowie Izráelſcy y ſynowie Judzcy od dźiećińſtwá ſwego to tylko cżynią / co jeſt złego przed ocżymá mojemi ; ſynowie / mówię / Izráelſcy tylko mię dráźnili ſpráwą rąk ſwojich / mówi Pán.
Zájiſte ná zápálcżywość moję / y ná gniew mój robi ſobie to miáſto ode dniá / którego je zbudowáli / áż do dniá tego / ták / że mi je przydźie oddálić od oblicża mego ;
A to dla wƺelkiej złośći ſynów Izráelſkich y ſynów Judzkich / którą popełniáli / pobudzájąc mię do gniewu / ſámi / królowie ich / kśiążętá ich / kápłáni ich y prorocy ich / jáko mężowie Judzcy / ták obywátele Jeruzálemſcy.
Obrácájąc śię do mnie tyłem á nie twárzą ; á gdy ich náucżám ráno wſtáwájąc y náucżájąc / wƺákże nie ſłucháją / áby przyjęli náukę.
Nádto náſtáwiáli obrzydliwośći ſwych w tym domu / który názwány jeſt od imienia mego / áby go ſplugáwili.
Nábudowáli / mówię / wyżyn Báálowi / które ſą w dolinie Ben Hennon / áby przenáƺáli przez ogień ſynów ſwojich y córki ſwoje Molochowi / choćiáżem im tego nie rozkázał / áni to wſtąpiło ná ſerce moje / áby kiedy cżynić mieli tę obrzydliwość / á do grzechu Judę przywodźić.
A teraz dla tego ták mówi Pán / Bóg Izráelſki / o tym mieśćie / o którem wy powiádáćie : Podáne jeſt w rękę królá Bábilońſkiego przez miecż / y głód / y mór :
Oto Ja zgromádzę ich ze wƺyſtkich źiem / do którychem ich wygnał w popędliwośći mojey y w gniewie moim / y w zápálcżywośći wielkiej / y przywiodę ich záś ná to miejſce / y ucżynię / áby bezpiecżnie mieƺkáli ;
Y będą ludem moim / á Ja będę Bogiem ich.
Y dam im ſerce jedno / y drogę jednę / áby śię mnie báli po wƺyſtkie dni / ták / áby śię im dobrze dźiáło / y ſynom ich po nich ;
Y ucżynię z nimi przymierze wiecżne / że śię nie odwrócę od nich / ábym im nie miał dobrze cżynić ; nádto bojáźń moję dam do ſercá ich / áby nie odſtępowáli odemnie.
Y będę śię weſelił z nich / dobrze im cżyniąc / gdyż ich wƺcżepię w tey źiemi wárownie / ze wƺyſtkiego ſercá mego y ze wƺyſtkiej duƺy mojey.
Bo ták mówi Pán : Jákom przywiódł ná ten lud to wƺyſtko wielkie złe / ták przywiodę ná nich to wƺyſtko dobre / o któremem z nimi mówił.
Tedy będą kupowáć rolę w tey źiemi / o którey wy powiádáćie : Spuſtoƺoná jeſt ták / że w niej niemáƺ áni cżłowieká áni bydlęćiá / podáná jeſt w rękę Cháldejcżyków.
Role zá pieniądze kupowáć będą / y zápiſem wárowáć / y piecżętowáć / y świádkámi oświádcżáć w źiemi Benjáminowej / y około Jeruzálemu / y w miáſtách Judzkich / jáko w miáſtách ná górách ták w miáſtách ná równinách / y w miáſtách ná południe / gdy záś przywrócę pojmánych ich / mówi Pán.
