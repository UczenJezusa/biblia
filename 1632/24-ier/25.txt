Słowo / które śię ſtáło do Jeremijáƺá przećiwko wƺyſtkiemu ludowi Judzkiemu roku cżwártego Joákimá / ſyná Jozyjáƺowego / królá Judzkiego / ( który jeſt rok pierwƺy Nábuchodonozorá / królá Bábilońſkiego ; )
Które mówił Jeremijáƺ prorok do wƺyſtkiego ludu Judzkiego / y do wƺyſtkich obywáteli Jeruzálemſkich mówiąc :
Od trzynáſtego roku Jozyjáƺá / ſyná Amonowego / królá Judzkiego / áż do dniá tego / już to przez dwádźieśćiá y trzy látá bywáło ſłowo Páńſkie do mnie / którem do was máwiał / ráno wſtáwájąc y opowiádájąc ; áleśćie nie ſłucháli.
Poſyłał też Pán do was wƺyſtkich ſług ſwojich proroków / ráno wſtáwájąc / y poſyłájąc / ( którycheśćie nie uſłucháli / ániśćie nákłonili uchá ſwego / ábyśćie ſłyƺeli ; )
Którzy mówili : Náwróććie śię teraz káżdy od złej drogi ſwojey / y od złośći ſpráw wáƺych ; á ták będźiećie mieƺkáli w tey źiemi / którą wam dał Pán / y ojcom wáƺym od wieku áż ná wieki.
A nie chodźćie zá bogámi cudzymi / ábyśćie im ſłużyli / y kłániáli śię im / áni mię gniewájćie ſpráwą rąk wáƺych / á Ja wam źle nie ucżynię.
Aleśćie mię nie uſłucháli / mówi Pán / ábyśćie mię pobudzáli do gniewu ſpráwą rąk ſwoich ná ſwe złe.
Przetoż ták mówi Pán zaſtępów : Dla tego / iżeśćie nie uſłucháli ſłów mojich /
Oto Ja poślę y pobiorę wƺyſtkie narody północne / mówi Pán / y do Nábuchodonozorá królá Bábilońſkiego / ſługi mego / y przywiodę je ná tę źiemię / y ná obywáteli jey / y ná te wƺyſtkie narody okolicżne / które do ƺcżętu wygłádzę / y położę je ná podźiw / y ná poświſtánie / y ná ſpuſtoƺenie wiecżne.
Y ſpráwię to / áby im zginął głos weſelá / y głos rádośći / głos oblubieńcá / y głos oblubienicy / głos żárn / y świátłość pochodni.
Y będźie tá wƺyſtká źiemiá ſpuſtoƺeniem / y zdumieniem / á ſłużyć będą te narody królowi Bábilońſkiemu śiedmdźieśiąt lat.
Ale potem / gdy śię wypełni śiedmdźieśiąt lat / náwiedzę ná królu Bábilońſkim y ná tym národźie / mówi Pán / niepráwość ich / y ná źiemi Cháldejſkiej / ták / że ją obrócę w puſtynie wiecżne.
A przywiodę ná tę źiemię wƺyſtkie ſłowá moje / którem mówił o niej / miánowićie to wƺyſtko / co nápiſano w tych kśięgách / cokolwiek prorokował Jeremijáƺ o wƺyſtkich národách.
Gdyż w niewolę podbići będą od narodów / tákże jáko y oni możnych / y od królów wielkich / tedy im oddam według ſpráw ich y według ucżynków rąk ich.
Bo ták rzekł Pán / Bóg Izráelſki / do mnie : Weźmij kubek winá tey popędliwośći z ręki mojey / á nápáwáj nim wƺyſtkie narody / do których Ja ćiebie poślę ;
Aby pili y potácżáli śię / owƺem / áby ƺáleli od oſtrzá miecżá / który Ja poślę miedzy nich.
Wźiąłem tedy kubek z ręki Páńſkiej / y nápoiłem wƺyſtkie one narody / do których mię Pán poſłał :
Jeruzálem / y miáſtá źiemi Judzkiej / y królów jey / y kśiążąt jey / ábym ich podał ná ſpuſtoƺenie / ná zdumienie / ná poświſtánie / y ná przeklęſtwo / jáko śię to dźiś okázuje.
Fáráoná też / królá Egipſkiego / y ſług jego / y kśiążąt jego / y wƺyſtek lud jego ;
Y to wƺyſtko poſpólſtwo / tákże wƺyſtkich króli źiemi Uz / y wƺyſtkich króli źiemi Filiſtyńſkiej / y Aƺkálon / y Gázę / y Akkáron / y oſtátek Azotu ;
Edomcżyków / y Moábcżyków / y ſynów Ammonowych ;
Y wƺyſtkich królów Tyrſkich / y wƺyſtkich królów Sydońſkich / y królów tey kráiny / która jeſt przy morzu ;
Dedáná y Temę / y Buzę / y wƺyſtkich / którzy mieƺkáją w oſtátnich kątách :
Y wƺyſtkich królów Arábſkich / y wƺyſtkich królów tego poſpólſtwá / które mieƺká ná puƺcży ;
Tákże wƺyſtkich królów Zymry y wƺyſtkich królów Elám / y wƺyſtkich królów Medſkich ;
Owƺem / wƺyſtkich królów północnych / bliſkich y dálekich / jednego jáko drugiego ; wƺyſtkie też króleſtwá źiemi / którekolwiek ſą ná oblicżu źiemi ; á król Seſák będźie pił po nich.
Y rzecż do nich : Ták mówi Pán zaſtępów / Bóg Izráelſki : Pijćie / á popijćie śię / owƺem zwrácájćie / y pádájćie ták / ábyśćie nie powſtáli dla miecżá / który Ja poślę miedzy was.
A jeſliby nie chćieli wźiąć kubká z ręki twojey / áby pili / tedy rzecżeƺ do nich : Ták mówi Pán zaſtępów : Koniecżnie pić muśićie.
Bo ponieważ ná to miáſto / które názwáne jeſt od imienia mego / Ja zácżynam przywodźić złe rzecży / á wybyśćie bez káránia być mieli? Nie będźiećie bez káránia ; bom Ja miecż przyzwał ná wƺyſtkich obywáteli tey źiemi / mówi Pán zaſtępów.
Przetoż ty prorokuj przećiwko nim te wƺyſtkie ſłowá / á mów do nich : Pán z wyſoká zárycży / á z mieƺkánia świętobliwośći ſwojey wydá głos ſwój / rycżąc zárycży z mieƺkánia ſwego ; krzyk pobudzájących śię / jáko tłocżących práſę / rozlegáć śię będźie przećiwko wƺyſtkim obywátelom tey źiemi.
Y przejdźie huk áż do końcżyn źiemi ; bo śię Pán rozpierá z tymi národámi / w ſąd ſám wchodźi ze wƺelkiem ćiáłem / niezbożnych podá pod miecż / mówi Pán.
Ták mówi Pán zaſtępów : Oto udręcżenie pójdźie z narodu do narodu / á wicher wielki powſtánie od końcżyn źiemi ;
Y będą pobići od Páná cżáſu onego od końcá źiemi áż do końcá źiemi ; nie będą ich płákáć / áni zbieráć / áni chowáć ; będą jáko gnój ná polu.
Nárzekájćie páſterze y wołájćie / á wálájćie śię w popiele / wy nájzácniejśi tey trzody! bo śię wypełniły dni wáƺe / zábićiá y rozproƺeniá wáƺego / y upádniećie jáko nacżynie drogie.
Y zginie ućiecżká páſterzom / á ujśćie nájzácniejƺym tey trzody.
Głos wołánia páſterzy / y nárzekánie nájzácniejƺych tey trzody ſłycháć będźie ; bo Pán ſpuſtoƺy páſtwiſká ich.
Y zágubione będą ſpokojne páſtwiſká dla zápálcżywośći gniewu Páńſkiego /
Który opuśći jáko lew jáſkinię ſwoję ; bo źiemiá ich przydźie ná ſpuſtoƺenie dla zápálcżywośći puſtoƺyćielá / y dla popędliwośći gniewu jego.
