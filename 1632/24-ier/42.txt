Potem przyſtąpili wƺyſcy kśiążętá wojſk / y Johánán / ſyn Káryjáƺowy / y Jáſánijáƺ / ſyn Hoſájáƺowy / y wƺyſtek lud od máłego áż do wielkiego ;
Y rzekli do Jeremijáƺá proroká : Niech przydźie proƺę prośbá náƺá przed oblicże twoje / á módl śię zá námi Pánu / Bogu twemu / zá wƺyſtek ten oſtátek ; bo nas máło zoſtáło z wielu / jáko to widźiƺ ocżymá twemi ;
A niech nam oznájmi Pán / Bóg twój / drogę / którąbyſmy chodźić / y cobyſmy cżynić mieli.
Y rzekł do nich Jeremijáƺ prorok : Słyƺę ; oto ja modlić śię będę Pánu / Bogu wáƺemu / według ſłów wáƺych / á cokolwiek wam Pán odpowie / oznájmię wam / nie zátáję nic przed wámi.
Oni záś rzekli do Jeremiaƺá : Niech będźie Pán miedzy námi świádkiem práwdźiwym y wiernym / jeſli nie ucżynimy według káżdego ſłowá / z którem ćię pośle Pán / Bóg twój / do nas.
Bądź dobrze bądź źle / głoſu Páná / Bogá náƺego / dla którego ćię poſyłámy do niego / uſłuchámy / áby śię nam dobrze dźiáło / gdy będźiemy ſłucháć głoſu Páná / Bogá náƺego.
A po wyśćiu dźieśięćiu dni / gdy śię ſtáło ſłowo Páńſkie do Jeremijáƺá /
Záwołał Johánáná / ſyná Kárejáƺowego / y wƺyſtkich kśiążąt wojſk / którzy z nim byli / y wƺyſtkiego ludu / od máłego áż do wielkiego /
Y rzekł do nich : Ták mówi Pán / Bóg Izráelſki / do któregośćie mię poſłáli / ábym przedłożył prośbę wáƺę przed oblicżem jego :
Jeſli śię náwróćićie / y zoſtániećie w tey źiemi / zájiſte pobuduję was / á nie zepſuję / y wƺcżepię was / á nie wykorzenię ; bo mi żál tego złego / którem wam ucżynił. Nie bójćież śię oblicża królá Bábilońſkiego / którego śię wy boićie ;
Nie bójćie śię go / mówi Pán ; bom jeſt z wámi / ábym was wybáwił y wyrwał was z ręki jego ;
Nádto zjednam wam łáſkę / áby śię zmiłował nád wámi / y dał śię wam wróćić do źiemi wáƺej.
Ale rzecżećieli : Nie zoſtániemy w tey źiemi / nie ſłuchájąc głoſu Páná / Bogá wáƺego /
A mówiąc : Żádną miárą ; ále do źiemi Egipſkiej pójdźiemy / gdźie nie ujrzymy wojny / áni głoſu trąby nie uſłyƺymy / á chlebá łáknąć nie będźiemy / y tám mieƺkáć będźiemy ;
Przetoż teraz ſłuchájćie ſłowá Páńſkiego / oſtátki Judzkie! Ták mówi Pán zaſtępów / Bóg Izráelſki : Jeſli wy upornie przy tym zoſtániećie / ábyśćie ƺli do źiemi Egipſkiej / á pójdźiećieli / ábyśćie tám mieƺkáli :
Tedy was pewnie miecż / którego śię boićie / tám w źiemi Egipſkiej dośćignie ; y głód / którego śię obáwiáćie / tám przydźie ná was w Egipćie / y tám pomrzećie.
Tákći śię ſtánie tym wƺyſtkim mężom / którzy śię ná to koniecżnie událi iść do Egiptu / áby tám pielgrzymowáli / że pomrą od miecżá / od głodu y od moru / á żáden z nich nie zoſtánie / áni kto ujdźie przed tym złem / które ja przywiodę ná nich.
Bo ták mówi Pán zaſtępów / Bóg Izráelſki : Jáko śię wyláłá popędliwość mojá y gniew mój ná obywáteli Jeruzálemſkich / ták śię wyleje zápálcżywość mojá ná was / gdy wnijdźiećie do Egiptu ; y będźiećie ná przeklinánie / y ná zdumienie / y ná złorzecżenie y ná háńbę / á nie oglądáćie więcey miejſcá tego.
Do wáſći mówi Pán / o oſtátki Judzkie! Nie wchodźćie do Egiptu ; wiedzćie wiedząc / ( bo śię dźiś oświádcżám przećiwko wam. )
Poniewáżeśćie zwiedli duƺe wáƺe / poſłáwƺy mię do Páná / Bogá wáƺego / mówiąc : Módl śię zá námi Pánu / Bogu náƺemu / á według wƺyſtkiego / cokolwiek powie Pán / Bóg náƺ / ták nam oznájmij / á my ucżynimy.
A gdy wam to dźiś oznájmuję / wƺákże nie ſłucháćie głoſu Páná / Bogá wáƺego / we wƺyſtkim / o co mię do was poſłał.
Przetoż mówię : Wiedzćie wiedząc / że miecżem / głodem y morem pomrzećie ná tym miejſcu / do którego prágniećie wnijść / ábyśćie tám pielgrzymowáli.
