Słowo / które śię ſtáło do Jeremijáƺá od Páná zá dni Joákimá / ſyná Jozyjáƺowego / królá Judzkiego / mówiąc :
Idź do domu Rechábitów / á mów z nimi / y wprowádź ich do domu Páńſkiego / do jedney komory / á dáj im pić wino.
Wźiąłem tedy z ſobą Jáſánijáƺá / ſyná Jeremijáƺowego / ſyná Chábázymijáƺowego / y bráći jego / y wƺyſtkich ſynów jego / y wƺyſtek dom Rechábitów /
Y wprowádźiłem ich do domu Páńſkiego / do komory ſynów Chánáná / ſyná Jegdálijáƺowego / mężá Bożego / która byłá podle komory kśiążęcey / która byłá nád komorą Mááſejáƺá / ſyná Sállumowego / ſtrzegącego progu.
Potem poſtáwiłem przed ſynámi domu Rechábitów cżáƺę pełną winá y kubki / y mówiłem do nich : Pijćie wino.
Którzy odpowiedźieli : Nie pijámy winá ; bo Jonádáb / ſyn Rechábowy / oćiec náƺ / zákázał nam / mówiąc : Nie pijájćie winá / wy y ſynowie wáśi áż ná wieki ;
A domu nie budujćie / y náśieniá nie rozśiewájćie / y winnicy nie ſádźćie / áni miewájćie ; ále w námiotách mieƺkájćie po wƺyſtkie dni wáƺe / ábyśćie żyli przez wiele dni ná oblicżu źiemi / w którey jeſteśćie przychodniámi.
Przetoż uſłucháliśmy głoſu Jonádábá / ſyná Rechábowego / ojcá náƺego / we wƺyſtkim / co nam rozkázał / żebyſmy nie pili winá po wƺyſtkie dni náƺe / my / żony náƺe / ſynowie náśi / y córki náƺe ;
Y żebyſmy nie budowáli domów ku mieƺkániu náƺemu / á winnicy / y roli / y żádnego śiewu nie mieli /
Ale ábyſmy mieƺkáli w námiotách ; y uſłucháliśmy / y ucżynimy według wƺyſtkiego / co nam rozkázał Jonádáb / oćiec náƺ.
A gdy przyćiągnął Nábuchodonozor Król Bábilońſki / do źiemi náƺej / rzekliśmy : Pójdźćie / á uſtąpmy do Jeruzálemu przed wojſkiem Cháldejſkiem / y przed wojſkiem Syryjſkiem ; á tákeśmy zoſtáli w Jeruzálemie.
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá / mówiąc :
Ták mówi Pán zaſtępów / Bóg Izráelſki : Idź / á rzecż mężom Judzkim y obywátelom Jeruzálemſkim : Y nie przyjmiećież ćwicżeniá / ábyśćie poſłuƺni byli ſłowom moim? mówi Pán.
Wáżne ſą ſłowá Jonádábá / ſyná Rechábowego / które przykázał ſynom ſwoim / żeby nie pili winá ; y nie piją go áż do dniá tego / bo poſłuƺni ſą przykázániu ojcá ſwego ; ále Ja mówię do was / ráno wſtáwájąc y mówiąc / á przećie jeſteśćie niepoſłuƺni.
Poſyłám też do was wƺyſtkich ſług ſwych / proroków ráno wſtáwájąc y poſyłájąc / áby mówili : Náwróććie śię już káżdy od złej drogi ſwej / á polepƺájćie ſpráw ſwojich / á nie náśládujćie bogów cudzych / áni im ſłużćie ; á ták mieƺkájćie w tey źiemi / którąm dał wam / y ojcom wáƺym ; áleśćie nie nákłonili uchá ſwego / ániśćie mię uſłucháli ;
Choć ſynowie Jonádábá / ſyná Rechábowego / doſyć ucżynili rozkázániu ojcá ſwego / które im przykázał ; ále ten lud nie jeſt mi poſłuƺny.
Przetoż ták mówi Pán / Bóg zaſtępów / Bóg Izráelſki : Oto Ja przywiodę ná Judę y ná wƺyſtkich obywáteli Jeruzálemſkich wƺyſtko złe / którem wyrzekł przećiwko im / przeto / żem mówił do nich / á nie ſłucháli / y wołáłem ich / á nie ozwáli mi śię.
A domowi Rechábitów rzekł Jeremijáƺ : Ták mówi Pán zaſtępów / Bóg Izráelſki : Dla tego / żeśćie poſłuƺni rozkázániu Jonádábá / ojcá wáƺego / y ſtrzeżećie wƺyſtkich przykázáń jego / á cżynićie według wƺyſtkiego / co wam rozkázał :
Przetoż ták mówi Pán zaſtępów / Bóg Izráelſki : Nie będźie wygłádzony mąż z rodu Jonádábá / ſyná Rechábowego / któryby ſtał przed oblicżem mojim po wƺyſtkie dni.
