Y uſłyƺał Sefátyjáƺ / ſyn Mátánowy / y Godolijáƺ / ſyn Fáſſurowy / y Juchál / ſyn Selemijáƺowy / y Fáſſur / ſyn Melchijáƺowy / ſłowá / które Jeremijáƺ mówił do wƺyſtkiego ludu / mówiąc :
Ták mówi Pán : Ktoby zoſtał w tym mieśćie / zginie od miecżá / od głodu y od moru ; ále ktoby wyƺedł do Cháldejcżyków / żyć będźie / á będźie mu duƺá jego zá korzyść / y żyw zoſtánie.
Ták mówi Pán : Pewnie podáne będźie to miáſto w ręce wojſká królá Bábilońſkiego / y weźmie je.
Przetoż rzekli oni kśiążętá do królá : Niech umrze ten mąż / ponieważ on oſłábiá ręce mężów wálecżnych / pozoſtáłych w tym mieśćie / y ręce wƺyſtkiego ludu / mówiąc do nich tákie ſłowá ; ábowiem ten mąż nie ƺuká pokoju ludowi temu / ále złego.
Tedy rzekł król Sedekijáƺ : Oto jeſt w ręce wáƺej ; bo król nic zgołá nie może przećiwko wam.
A ták wźięli Jeremijáƺá / który był w śieni ſtráży / y wrzućili go do dołu Málchyjáƺá / ſyná królewſkiego / y ſpuśćili Jeremijáƺá po powrozách ; á w tym dole nie było nic wody / tylko błoto / ták tonął Jeremijáƺ w onem błoćie.
Ale gdy uſłyƺał Ebedmelech Murzyn / dworzánin / który był w domu królewſkim / że Jeremijáƺá podáno do dołu / ( á król śiedźiał w bramie Benjáminowej. )
Wnet wyƺedł Ebedmelech z domu królewſkiego / y rzekł do królá / mówiąc :
Królu / pánie mój! źle ucżynili ći mężowie wƺyſtko / co ucżynili Jeremijáƺowi prorokowi / że go wrzućili do tego dołu ; boćby był umárł ná pierwƺem miejſcu od głodu / ponieważ już niemáƺ żádnego chlebá w mieśćie.
Przetoż rozkázał król Ebedmelechowi Murzynowi / mówiąc : Weźmij z ſobą ſtąd trzydźieśći mężów / á wyćiągnij Jeremijáƺá proroká z tego dołu / niżby umárł.
Tedy wźiął Ebedmelech onych mężów z ſobą / y wƺedł do domu królewſkiego pod ſkárbnicę / y nábrał z támtąd ſtárych ƺmáćiſk podártych / ƺmáćiſk / mówię / zbótwiáłych / które ſpuśćił do Jeremijáƺá do onego dołu po powrozách.
Y rzekł Ebedmelech Murzyn do Jeremijáƺá : Nuże podłóż te ſtáre podárte y zbótwiáłe ƺmáćiſká pod páchy rąk twojich pod powrozy ; y ucżynił ták Jeremijáƺ.
Wyćiągnęli tedy Jeremijáƺá powrozámi / y dobyli go z onego dołu ; y śiedźiał Jeremijáƺ w śieni ſtráży.
Tedy poſłał król Sedekijáƺ y wźiął Jeremijáƺá proroká do śiebie do trzećich drzwi / które były przy domu Páńſkim. Y rzekł król Jeremijáƺowi : Spytám ćię o jednę rzecż / nie táj nic przedemną.
Y rzekł Jeremijáƺ do Sedekijáƺá : Jeſlić co powiem / pewnie mię zábijeƺ? A jeſlić co porádzę / nie uſłucháƺ mię.
Tedy przyśiągł król Sedekijáƺ Jeremijáƺowi potájemnie / mówiąc : Jáko żyje Pán / który nam tę duƺę ſtworzył / że ćię nie zábiję / áni ćię wydám w rękę mężów tych / którzy ƺukáją duƺy twojey.
Y rzekł Jeremijáƺ do Sedekijáƺá : Ták mówi Pán / Bóg zaſtępów / Bóg Izráelſki : Jeſli dobrowolnie wynijdźieƺ do kśiążąt królá Bábilońſkiego / tedy żyć będźie duƺá twojá / á to miáſto nie będźie ſpálone ogniem ; á ták żyw zoſtánieƺ ty y dom twój ;
Ale jeſli nie wynijdźieƺ do kśiążąt królá Bábilońſkiego / pewnie będźie podáne to miáſto w rękę Cháldejcżyków / y ſpálą je ogniem / á y ty nie ujdźieƺ ręki ich.
Tedy rzekł król Sedekijáƺ do Jeremijáƺá : Bárzo śię boję Żydów / którzy poućiekáli do Cháldejcżyków / bym ſnadź nie był wydány w rękę ich / á ƺydźiliby ze mnie.
Y rzekł Jeremijáƺ : Nie wydádzą. Słucháj proƺę głoſu Páńſkiego / któryć ja opowiádám / á będźieć dobrze / y żyć będźie duƺá twojá ;
A jeſli śię będźieƺ zbrániał wynijść / tedy to jeſt ſłowo / które mi Pán pokázał :
Że oto wƺyſtkie niewiáſty / które zoſtáły w domu królá Judzkiego / będą wywiedźione do kśiążąt królá Bábilońſkiego / á te ſáme rzeką : Námówili ćię / y otrzymáli to ná tobie przyjáćiele twoi / że ulgnęły w błoćie nogi twoje / y cofnęły śię názád.
Wƺyſtkie tákże żony twoje y ſynów twojich wywiodą do Cháldejcżyków / y ty ſám nie ujdźieƺ ręki ich / owƺem ręką królá Bábilońſkiego będźieƺ pojmány / y to miáſto ſpálą ogniem.
Tedy rzekł Sedekijáƺ do Jeremijáƺá : Niecháj nikt nie wie o tym / ábyś nie umárł ;
A jeſliby kśiążętá uſłyƺáwƺy / żem mówił z tobą / przyƺli do ćiebie / y rzeklić : Powiedz nam / proƺę / coś mówił z królem / nie táj przed námi / á nie zábijemy ćię ; co z tobą król mówił?
Tedy im rzecżeƺ : Przełożyłem prośbę moję przed królem / áby mię záś nie kázał odwieść do domu Jonátánowego / żebym tám nie umárł.
A zeƺli śię wƺyſcy kśiążętá do Jeremijáƺá / y pytáli go ; który im powiedźiał według tego wƺyſtkiego / jáko mu był król rozkázał. A ták milcżąc odeƺli od niego / gdyż śię to było nie ogłośiło.
A Jeremijáƺ śiedźiał w śieni ſtráży áż do onego dniá / którego wźięto Jeruzálem / gdźie był / gdy dobywano Jeruzálemu.
