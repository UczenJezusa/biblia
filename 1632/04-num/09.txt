Y rzekł PAN do Mojzeƺá ná puƺczy Synáj / roku wtórego po wyśćiu ich z źiemie Egipſkiey / mieśiącá pierwƺego / mówiąc :
Niech obchodzą Synowie Izráelſcy święto Prześćia / czáſu náznáczonego :
Czternaſtego dniá mieśiącá tego / miedzy dwiemá wieczorámi obchodźić je będźiećie / czáſu náznáczonego : według wƺyſtkich obrzędów jego / y według wƺyſtkich Ceremonii jego / obchodźić je będźiećie.
Mówił tedy Mojzeƺ do Synów Izráelſkich / áby obchodźili święto Prześćia.
Y obchodźili Święto Prześćiá / pierwƺego mieśiącá czternaſtego dniá / miedzy dwiemá wieczorámi ná puƺczy Synáj : według wƺyſtkiego / jáko rozkazał PAN Mojzeƺowi / ták uczynili Synowie Izráelſcy.
Y byli niektórzy ludźie którzy śię byli ſplugáwili nád umárłym człowiekiem / y nie mogli obchodźić Świętá Prześćia dniá onego : Tedy przyſtąpili do Mojzeƺá y do Aároná w tenże dźień /
Y rzekli oni ludźie do niego / zmázáliſmy śię nád umárłym / y nie będźież nam wolno oddáć Ofiáry PAnu / czáſu náznáczonego / weſpół z Synmi Izráelſkimi?
Którym odpowiedźiał Mojzeƺ : Poſtójćie / áż uſłyƺę co rozkaże PAN o was.
Tedy rzekł PAN do Mojzeƺá / mówiąc :
Powiedz Synom Izráelſkim / y rzecz : Jeſliby śię kto zmázał nád umárłym / álboby ná drodze dálekiey był / ták z was / jáko y z potomſtwá wáƺego / <i>przećię</i> będźie odpráwował Święto Prześćiá PAnu :
Mieśiącá wtórego / czternaſtego dniá / miedzy dwiemá wieczorámi odpráwowáć je będą z przáſnemi Chleby / y z gorzkimi źioły / jeść je będą :
Nie zoſtáwią <i>nic</i> z niego do jutrá / y kośći nie złámią w nim : według wƺyſtkiego poſtánowienia / Świętá Prześćia / odpráwowáć je będą.
Ale człowiek któryby był czyſty / á nie byłby w drodze / y nie obchodźiłby Świętá Prześćia / tedy duƺá oná wykorzenioná będźie z ludu ſwego / bo Ofiáry PAńſkiey nie odpráwował / czáſu náznáczonego : grzech ſwój ponieśie on człowiek.
A jeſliby przychodźień mieƺkájący miedzy wámi / obchodźił Święto Prześćia PAnu / według uſtáwy świętá prześćia / y według obrzędów jego / obchodźić je będźie : uſtáwá jedná będźie wam / y przychodniowi / y zrodzonemu w źiemi.
Dniá tedy onego / którego wyſtáwiony był Przybytek / Obłok okrył Przybytek / nád Namiotem Świádectwá / á wieczór bywáło nád Przybytkiem / jáko widzenie ogniá / áż do poránku.
Ták bywáło uſtáwicznie : <i>we dnie</i> okrywał go Obłok / á <i>jáko</i> widzenie Ogniá w nocy.
A gdy śię podnaƺał Obłok od Namiotu / tedy śię ruƺáli Synowie Izráelſcy : á gdźiekolwiek ſtawał Obłok / támże ſtánowili Obóz Synowie Izráelſcy.
Ná rozkazánie PAńſkie ćiągnęli Synowie Izráelſcy y ná rozkazánie PAńſkie ſtánowili Obóz po wƺyſtkie dni których zoſtawał Obłok nád Przybytkiem / <i>y oni</i> leżeli Obozem.
A gdy trwał Obłok on nád Przybytkiem przez wiele dni / tedy odpráwowáli Synowie Izráelſcy ſtraż PAnu / á nie ruƺáli śię.
Ale gdy nie długo trwał Obłok nád Przybytkiem / ná rozkazánie PAńſkie ſtánowili Obóz / y ná rozkazánie PAńſkie / ćiągnęli :
A gdy bywał Obłok od wieczorá áż do poránku / á podnośił śię záś poránu / tedy ćiągnęli : ták we dnie jáko y w nocy / gdy śię podnióſł Obłok ćiągnęli.
A jeſli przez dwá dni / álbo przez mieśiąc / álbo <i>też</i> przez rok trwał Obłok nád Przybytkiem / zoſtawájąc nád nim / Obozem leżeli Synowie Izráelſcy / y nie ruƺáli śię : Ale gdy śię on podnośił / <i>y oni</i> śię ruƺáli.
Ná rozkazánie PAńſkie ſtánowili Obóz / y ná rozkazánie PAńſkie ćiągnęli : ſtraż PAńſką trzymájąc / jáko im PAN rozkazał przez Mojzeƺá.
