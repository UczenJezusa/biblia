Tedy śię zbuntował Kore / Syn Iſáárá / Syná Káátowego / Syná Lewiego / tákże Dátán y Abiron / Synowie Eliábowi / y Hon / Syn Peletów z Synów Rubenowych.
Y powſtáli przećiw Mojzeƺowi / á <i>z nimi</i> mężów z Synów Izráelſkich / dwieśćie y pięćdźieśiąt / Kśiążętá miedzy ludem / których do rády przyzywano / ludźie zacni.
Ci zebráwƺy śię przećiw Mojzeƺowi / y przećiw Aáronowi / rzekli im : wiele <i>to</i> ná was : ponieważ wƺyſtek ten lud / wƺyſcy ſą święći / á w pośrzodku nich <i>jeſt</i> PAN / przeczże śię wy wynośićie nád zgromádzeniem PAńſkim?
Co gdy uſłyƺał Mojzeƺ / upadł ná oblicże ſwoje :
Y rzekł do Korego / y do wƺyſtkiey roty jego / mówiąc / ráno pokaże PAN kto <i>jeſt</i> jego / y <i>kto jeſt</i> święty / y kto ma przyſtępowáć przedeń : bo kogo ſobie wybrał / temu rozkaże przyſtąpić do śiebie.
To <i>tedy</i> uczynićie : Weźmiećie ſobie kádźilnice / <i>ty</i> Kore / y wƺyſtká rotá twojá /
Y nákładƺy w nie Ogniá / włożćie nań kádźidłá przed PAnem jutro : y ſtánie śię / <i>że</i> kogokolwiek obierze PAN / ten <i>będźie</i> świętym : Wiele <i>to</i> ná was Synowie Lewiego.
Nád to rzekł Mojzeƺ do Korego : Słuchajćie proƺę Synowie Lewiego :
Yzali wam <i>to</i> máło / że was oddźielił Bóg Izráelſki / od zgromádzenia Izráelſkiego / ábyśćie przyſtępowáli do niego / ku odpráwowániu uſługi w Przybytku PAńſkim / á yżbyśćie ſtali przed zgromádzeniem / y ſłużyli mu /
Y przyjął ćiebie / y wƺyſtkę bráćią twoję / Syny Lewiego z tobą / że jeƺcze ƺukaćie kápłáńſtwá?
Dla tegoż / ty y wƺyſtká rotá twojá / zbuntowáliśćie śię przećiw PAnu : bo Aáron cóż jeſt / żeśćie ƺemráli przećiw niemu?
Tedy poſłał Mojzeƺ / áby záwołano Dátáná / y Abironá Synów Eliábowych : którzy odpowiedźieli : Nie pójdźiemy.
Izali máło <i>ná tym,</i> żeś nas wywiódł z źiemie opływájącey mlekiem y miodem / ábyś nas pomorzył ná tey puƺczy / że jeƺcze chceƺ mieć nád námi zwierzchność / y nam rozkázowáć?
Ku temu / do źiemie opływájącey mlekiem y miodem / nie wprowádźiłeś nas / ániś nam dał w dźiedźictwo pól y winnic : Izali oczy tym mężom wyłupić chceƺ? Nie pójdźiemy.
Tedy śię rozgniewał Mojzeƺ bárzo / y rzekł do PAná : Nie pátrz ná Ofiárę ich : ni jednego oſłá nie wźiąłem od nich / ánim co złego komu z nich uczynił.
Potym rzekł Mojzeƺ do Korego : Ty y wƺyſtká rotá twojá / ſtawćie śię przed PAná jutro : ty / y oni / y Aáron :
A wźiąwƺy káżdy kádźilnicę ſwoję / włóżćie w nię kádźidłá : y ſtawćie śię przed PAná káżdy z kádźilnicą ſwoją / dwieśćie y pięćdźieśiąt kádźilnic : y ty / y Aáron káżdy z kádźilnicą ſwoją.
Wźiął tedy káżdy kádźilnicę ſwoję / á włożywƺy w nię Ogniá / nákłádli w nię kádźidłá : y ſtánęli u drzwi Namiotu zgromádzenia / Mojzeƺ y Aáron.
Ale Kore <i>już był</i> zebrał przećiwko nim wƺyſtkę rotę / do drzwi Namiotu zgromádzenia : tedy chwałá PAńſka ukazáłá śię wƺyſtkiemu ludowi.
Y rzekł PAN do Mojzeƺá y do Aároná / mówiąc :
Odłącżćie śię z pośrzodku zebránia tego / ábym je nagle zátráćił.
Lecż oni upádli ná Oblicża ſwe / y rzekli : O Boże! Boże Duchów / <i>y</i> wƺelkiego ćiáłá! cżłowiek jeden zgrzeƺył / á ná wƺyſtekże lud gniewáć śię będźieƺ?
Tedy rzekł PAN do Mojzeƺá mówiąc :
Rzecz do zgromádzenia / á mów : Odſtąpćie od namiotu Korego / Dátáná y Abironá.
A wſtawƺy Mojzeƺ ƺedł do Dátáná y Abironá / y ƺli zá nim ſtárƺy Izráelſcy.
Y rzekł do zgromádzenia / mówiąc odſtąpćie / proƺę / od namiotów mężów tych niepobożnych / áni śię dotykajćie wƺyſtkiego co ich jeſt : byśćie ſnadź nie zginęli we wƺyſtkich grzechách ich.
Y odſtąpili od namiotu Korego / Dátáná y Abironá zewƺąd. Ale Dátán y Abiron wyƺedƺy / ſtali u drzwi namiotów ſwojich / y żony ich / y Synowie ich / y málucżcy ich.
Tedy rzekł Mojzeƺ : Potym poznaćie / że mię PAN poſłał / ábym cżynił te wƺyſtkie ſpráwy / á że nic <i>z domyſłu</i> ſercá ſwego <i>nie czynię</i> :
Jeſliże ták jáko inni ludźie umieráją / pomrą ći / á <i>zwykłym</i> innych ludźi karániem karáni będą / nie poſłał mię PAN.
Ale jeſliż co nowego ucżyni PAN / że otworzy źiemiá uſtá ſwe / y pozrzre je / y wƺyſtkiego co máją / y zſtąpią żywo do Piekłá / tedy poznaćie / że rozdraźnili mężowie ći PAná.
Y ſtáło śię gdy przeſtał mówić wƺyſtkich tych ſłów / że śię rozſtąpiłá źiemiá pod nimi /
A otworzywƺy źiemiá páƺcżękę ſwoję / pożárłá je / y domy ich / ze wƺyſtkimi ludźmi którzy byli przy Korem / y wƺyſtkie májętnośći <i>ich</i>.
Y zſtąpili oni ze wƺyſtkim co mieli / żywo do Piekłá / y okryłá je źiemiá / y poginęli z pośrzodku zgromádzenia.
Wƺyſcy záś Izráelitowie którzy <i>byli</i> około nich / ućiekáli ná krzyk ich / bo mówili : by ſnadź nie pożárłá <i>y</i> nas źiemiá.
Wyƺedł tákże Ogień od PAná / y ſpalił onych dwieśćie y pięćdźieśiąt mężów którzy ofiárowáli kádzenie.
Potym rzekł PAN do Mojzeƺá / mówiąc :
Rzecż do Eleázárá Syná Aáronowego / Kápłaná / áby pozbierał kádźilnice z onego pogorzeliſká / á Ogień y tám y ſám niech rozrzući : bo ſą poświęcone :
A z kádźilnic tych którzy zgrzeƺyli przećiwko duƺam ſwym / rozbiwƺy je ná bláchy / niech obije Ołtarz : bo iż w nich ofiárowáli przed PAnem / poświęcone ſą / przetoż będą ná znák Synom Izráelſkim.
Tedy pozbierał Eleázár Kápłan one miedźiáne kádźilnice / w których ofiárowáli oni popaleni : y rozbito je ná bláchy / ná Obićie Ołtarzá /
Ná pámiątkę Synom Izráelſkim / áby nie przyſtępował żaden Obcy / któryby nie był z naśienia Aáronowego / do odpráwowánia kádzenia przed PAnem / áby mu śię nie ſtáło jáko Koremu / y jáko roćie jego : jáko mu był powiedźiał PAN przez Mojzeƺá.
Y ƺemráło wƺyſtko zgromádzenie Synów Izráelſkich názájutrz / przećiwko Mojzeƺowi / y przećiwko Aáronowi / mówiąc : wyśćie przycżyną śmierći ludu PAńſkiego.
Y ſtáło śię / gdy śię zbierał lud przećiw Mojzeƺowi / y przećiw Aáronowi / że pojrzeli ku Namiotowi zgromádzenia : á oto okrył go Obłok / y okazáłá śię chwałá PAńſka.
Y przyƺedł Mojzeƺ z Aáronem przed Namiot zgromádzenia.
Y rzekł PAN do Mojzeƺá / mówiąc :
Wynidźćie z pośrzodku zgromádzenia tego / á wytrácę je w oká mgnieniu : Y upádli ná oblicza ſwoje.
Potym rzekł Mojzeƺ do Aároná : Weźmi kádźilnicę / á włóż w nię ogniá z Ołtarzá / włóż też Kádźidłá / á ná tych miaſt idź do zgromádzenia / y oczyść je : boć już wyƺłá popędliwość od twarzy PAńſkiej / <i>á już</i> śię záczęło karánie.
Wźiął tedy Aáron <i>kádźilnicę,</i> jáko mu rozkazał Mojzeƺ / y przybieżał w pośrzodek zgromádzenia / á oto <i>już</i> śię byłá záczęłá plagá w ludu : y uczyniwƺy kádzenie ocżyśćił lud.
Y ſtánął Aáron miedzy umárłymi / y miedzy żywemi / á záhámowána jeſt plagá.
A było umárłych od oney plagi / cżternaśćie tyśięcy / y śiedm ſet / oprócz onych którzy pomárli z przycżyny Korego.
Zátym śię wróćił Aáron do Mojzeƺá / ku drzwiom Namiotu zgromádzenia / gdy plagá záhámowána byłá.
