Tedy wzruƺywƺy śię wƺyſtko mnóſtwo krzyczeli / y płákał lud przez onę noc.
Y ƺemráli przećiwko Mojzeƺowi / y przećiwko Aáronowi / wƺyſcy Synowie Izráelſcy : y mówiło do nich wƺyſtko mnóſtwo : O byſmy byli pomárli w źiemi Egipſkiey / álbo ná tey puƺczy!
O byſmy byli pomárli! Czemuż wżdy PAN prowádźi nas do tey źiemie / ábyſmy pádli od mieczá? żony náƺe / y dźiatki náƺe / áby były ná łup? Izali nam nie lepiey wróćić śię do Egiptu?
Y mówili miedzy ſobą : Poſtánówmy ſobie wodzá / á wróćmy śię do Egiptu.
Tedy upadł Mojzeƺ y Aáron ná oblicze ſwoje / przed wƺyſtkim zgromádzeniem Synów Izráelſkich.
A Jozue ſyn Nunów / y Káleb ſyn Jefunów / <i>którzy</i> ƺpiegowáli źiemię rozdárli ƺáty ſwoje :
Y rzekli do wƺyſtkiego zgromádzenia Synów Izráelſkich / mówiąc : Ziemiá którąſmy przeƺli y przeƺpiegowáli ją / źiemiá jeſt bárzo dobrá :
Będźieli nam PAN miłośćiw / tedy nas wprowádźi do tey źiemie / á da ją nam / źiemię tę / która opływa mlekiem y miodem :
Jedno PAnu nie bądźćie odpornymi / áni śię wy bójćie ludu oney źiemie / bo jáko chleb pojeść je <i>możemy :</i> odſtąpiłá obroná ich od nich / ále PAN jeſt z námi : nie bójćież śię ich.
Y mówiło wƺyſtko zgromádzenie / áby je ukámionowano : Ale Chwałá PAńſka okazáłá śię nád Namiotem zgromádzenia wƺyſtkim Synom Izráelſkim.
Y rzekł PAN do Mojzeƺá / dokądże mię draźnić będźie ten lud? Y dokądże mi wierzyć nie będą? Dla tych wƺyſtkich znáków którem cżynił miedzy nimi?
Poráżę je morem / y rozproƺę je / á ćiebie ucżynię w Naród wielki / y możnieyƺy / niż ten jeſt.
Ale rzekł Mojzeƺ do PAná : Oto uſłyƺą Egipcżánie / z których pośrzodku wywiodłeś mocą ſwoją ten lud :
Y mówić będą z obywátelmi źiemie tey : ( <i>bo</i> ſłyƺeli żeś ty PAnie był w pośrzodku ludu tego / żeś okiem w oko widźiány był / o PAnie / á obłok twóy ſtawał nád nimi / á iż w ſłupie obłokowym chodźiłeś przed nimi we dnie / á w ſłupie ogniſtym / w nocy. )
Gdybyś tedy pobił lud ten wƺyſtek áż do jednego / rzekliby Pogánie / którzy o twey ſławie ſłycháli / mówiąc :
Iż nie mógł wprowádźić PAN ludu tego do źiemie / o którą im przyśiągł / przeto je pobił ná puƺczy.
A ták teraz niech proƺę uwielbiona będźie moc PAńſka / jákoś rzekł / mówiąc :
PAN nie rychły ku gniewowi / á wielki w miłośierdźiu znoƺąc niepráwość y przeſtępſtwo / który winnego nie cżyni niewinnym / karząc niepráwość Ojców w syniech / do trzećiego y do czwartego pokolenia :
Odpuść proƺę niepráwość ludu tego / według wielkośći miłośierdźia twego / ták jákoś odpuƺcżał ludowi temu / z Egiptu áż do tąd.
Tedy rzekł PAN : Odpuśćiłem według ſłowá twego.
A wƺákże jáko ja żyję / y napełniona jeſt chwałą PAńſką wƺyſtká źiemiá :
Ták wƺyſcy / którzy widźieli chwałę moję / y znáki moje / którem czynił w Egipćie / y ná puƺcży / á kuśili mię już po dźieśięć kroć / áni byli poſłuƺni głoſowi memu :
Nie oglądáją źiemie tey / o którąm przyśiągł Ojcom ich : á żaden z tych / którzy mię draźnili / nie oglądáją jey :
Ale ſługę mego Kálebá / ( gdyż był w nim duch inákƺy / y trwał ſtátecżnie przy mnie / ) wprowádzę do źiemie / do którey chodźił : á naśienie jego odźiedźiczy ją :
Ale ponieważ Amálekytá y Chánánejczyk mieƺkáją w dolinie : <i>przetoż</i> jutro obróććie śię / á idźćie ná puƺczą / drogą ku morzu czerwonemu.
Nad to rzekł PAN do Mojzeƺá y do Aároná / mówiąc :
Y dokądże znośić mam ten zły lud / który ƺemrze przećiwko mnie? <i>długoż</i> ƺemránia Synów Izráelſkich / którzy ƺemrzą przećiwko mnie / ſłucháć będę?
Mów do nich : żyję ja / mówi PAN / że jákośćie mówili w uƺy moje ták uczynię wam :
Ná tey puƺczy polegą ćiáłá wáƺe / y wƺyſcy policzeni wáƺy / według wƺyſtkiey licżby wáƺey / od dwudźieſtego roku y wyżey / którzyśćie ƺemráli przećiwko mnie :
A wy nie wnidźiećie do źiemie tey / o którąm podnióſł rękę moję / ábym ją wam dał ná mieƺkánie / okrom Kálebá Syná Jefunowego / y Jozue Syná Nunowego :
A dźiatki wáƺe / o którycheśćie mówili / że będą ná łup / te wprowádzę / y oglądáją tę źiemię / którąśćie wy wzgárdźili :
Ale trupy wáƺe / wáƺe trupy mówię / polegą ná tey puƺczy :
A Synowie wáƺy będą śię tułáli po tey puƺczy przez czterdźieśći lat / y ponioſą <i>karanie</i> zá cudzołóſtwá wáƺe / áż wyginą trupy wáƺe ná puƺczy :
Według licżby dni / w którycheśćie ƺpiegowáli źiemię / <i>to jeſt</i> czterdźieśći dni / dźień káżdy zá rok <i>licżąc</i> ponieśiećie niepráwośći wáƺe czterdźieśći lat / y poznaćie <i>pomſtę</i> ſwego odſtąpienia ode mnie.
<i>JA PAN</i> mówiłem / że to ucżynię / temu wƺyſtkiemu zgromádzeniu złemu / które śię ſpiknęło przećiwko mnie : ná tey puƺczy poginą / y tu pomrą.
Oni tedy mężowie / których ſłał Mojzeƺ ná przeƺpiegowánie źiemie / którzy wróćiwƺy śię pobudźili do ƺemránia / przećiwko niemu wƺyſtek lud / puƺczájąc złą ſławę o źiemi oney :
Pomárli mężowie oni / którzy puƺczáli ſławę złą o źiemi / ſrogą plagą / przed PAnem.
Ale Jozue Syn Nunów / y Káleb Syn Jefunów / zoſtáli żywi z mężów onych / którzy chodźili ku przeƺpiegowániu źiemie.
Y opowiedźiał Mojzeƺ te ſłowá wƺyſtkim Synom Izráelſkim / y płákał lud bárzo.
Tedy ráno wſtawƺy / wſtąpili ná wierzch góry / mówiąc : Oto my pójdźiemy ná to miejſce / o którym nam PAN powiedźiał : boſmy zgrzeƺyli.
Ale im powiedźiał Mojzeƺ : Przecżże wy przeſtępujećie ſłowo PAńſkie? to śię wam nie náda.
Nie chodźćie / bo niemáƺ PAná miedzy wámi : ábyśćie nie byli pobići od nieprzyjaćiół wáƺych :
Bo Amálekytá y Chánáneycżyk <i>tuż</i> przed wámi ſą / y polężećie od miecżá : Bo dla tego / żeśćie śię odwróćili od PAná / nie będźie PAN z wámi.
A oni przećię kuśili śię wniść ná wierzch góry : lecż ſkrzyniá przymierza PAńſkiego / y Mojzeƺ / nie odchodźili od Obozu.
Tedy zſtąpił Amálekytá y Chánáneycżyk / mieƺkájący ná oney górze / á poráźili je / y gonili je áż do Hormy.
