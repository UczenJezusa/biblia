Potym ruƺył śię lud z Háſſerotu / á położyli śię obozem ná Puƺczy Fárán.
Y rzekł PAN do Mojzeƺá / mówiąc :
Pośli męże którzyby przeƺpiegowáli źiemię Chánánejſką / którą ja dawam Synom Izráelſkim : po jednym mężu / z káżdego pokolenia poślećie / którzyby byli przedniejƺy miedzy nimi.
Wyſłał je tedy Mojzeƺ z Puƺczy Fárán / według rozkazánia PAńſkiego / á oni mężowie wƺyſcy / byli co przednieyƺy / z Synów Izráelſkich.
A teć ſą imioná ich : Z pokolenia Ruben / Sámuáh / ſyn Záchurów.
Z pokolenie Symeonowego / Sáfát / ſyn Hurów.
Z pokolenia Judy / Káleb / ſyn Jefonów.
Z pokolenia Iſáſchár / Ighál / ſyn Józefów.
Z pokolenia Efráimowego / Ozee / ſyn Nunów.
Z pokolenia Beniáminowego / Fálty / ſyn Ráfuów.
Z pokolenia Zábulon / Gediel / ſyn Sodego.
Z pokolenia Józefowego / to jeſt / z Potomſtwá Mánáſſeſſowego / Gáddy / ſyn Suſego.
Z pokolenia Dán / Ammiel / ſyn Gemmálego.
Z pokolenia Aſer / Sethur / ſyn Micháelów.
Z pokolenia Neftáli / Náhbi / ſyn Wáfſego.
Z pokolenia Gád / Guel / ſyn Máchego.
Teć <i>ſą</i> imioná mężów / które poſłał Mojzeƺ ná przeƺpiegowánie źiemie : y názwał Mojzeƺ Ozeaƺá ſyná Nunowego / Jozue.
A poſyłájąc je Mojzeƺ ná przeƺpiegowánie źiemie Chánáneyſkiey / mówił do nich : Idźćie w tę ſtronę ku Południowi / á wnidzćie ná górę.
Y oglądájćie źiemię jáka jeſt / y lud który mieƺka w niey / jesli jeſt mocny czyli mdły : jesli ich máło / czyli wiele.
Jáka też <i>jeſt</i> źiemiá / w którey mieƺkáją / jesli dobra / czyli zła : y co ſą zá miáſtá w których mieƺkáją : jesli w namiećiech / czyli w obronnych miejſcách :
Tákże co zá źiemiá / jeſli urodzájna / czyli nie płodna : jeſli w niey ſą drzewá / czyli nie / á bądźćie mężnego ſercá / y przynieśćie <i>nam</i> Owocu tám tey źiemie : á było to w ten czás / gdy śię winá doſtawáły.
Y ƺli / á przeƺpiegowáli źiemię / od puƺczy Syn / áż do Rochob / którędy chodzą do Emát.
A idąc ku południowi / przyƺli áż do Hebron / gdźie byli / Achimán / Seſáy y Tholmáy / ſynowie Enákowi. A Hebron śiedm lat było zbudowáne przed Soán Egipſkim.
Przyƺli potym áż do rzeki Eſchol / y urznęli tám gáłąź z gronem jednym jágód winnych / y nieśli ją ná drążku / dwá tákże gránatowe jábłká y figi.
Y názwano miejſce ono Náchál Eſchol / od groná winnego / które tám byli urznęli Synowie Izráelſcy.
Zátym wróćili śię od onego ƺpiegowánia źiemie / po wyśćiu czterdźieſtu dni.
A wróćiwƺy śię przyƺli do Mojzeƺá y do Aároná / y do wƺyſtkiego zgromádzenia Synów Izráelſkich / ná puƺczą Fárán / <i>która jeſt</i> w Kádes : y dáli im y wƺyſtkiemu poſpólſtwu ſpráwę / ukazawƺy im Owoc oney źiemie.
A powiádáli im / mówiąc : Przyƺliſmy do źiemie / do któreyś nas był poſłał / która w prawdźie opływa mlekiem y miodem / á oto ten jeſt owoc jey :
Tylko że mocny jeſt lud / który mieƺka w oney źiemi / miáſtá tákże obronne ſą <i>y</i> bárzo wielkie : do tego y Syny Enákowe / támeſmy widźieli.
Amálek mieƺka w źiemi ná Południe : á Hettejczyk / y Jebuzejczyk / y Amorrejczyk / mieƺka ná górách : Chánánejczyk záś mieƺka nád morzem / y nád brzegiem Jordanu.
Y hámował Káleb lud <i>ƺemrzący</i> przećiw Mojzeƺowi / y mówił / pódźmy á pośiądźmy źiemię / bo ją pewnie ſobie podbijemy.
Ale mężowie oni / którzy z nim chodźili / rzekli : Nie będźiem mogli ćiągnąć przećiw támtemu ludowi / bo mocniejƺy jeſt nád nas.
Y zgánili onę źiemię którą byli przeƺpiegowáli / Synom Izráelſkim / mówiąc : Ziemiá tá którąsmy przeƺli / y przeƺpiegowáli ją / jeſt źiemiá pożyrájąca obywátele ſwoje : á lud wƺyſtek / któryſmy w niey widźieli lud <i>jeſt</i> wyſokiego wzroſtu.
Támesmy też widźieli Obrzymów / ſyny Enákowe / z rodu Obrzymów : y zdáliſmy śię ſobie <i>przy nich</i> jáko ƺáráńcza / tákimiż zdálismy śię y onym.
