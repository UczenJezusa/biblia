Potym rzekł PAN do Mojzeƺá / mówiąc :
Pomśći śię krzywdy Synów Izráelſkich / nád Mádiánitámi / y potym przyłączon będźieƺ do ludu twego.
Tedy rzekł Mojzeƺ do ludu / mówiąc : wyprawćie z pośrzodku śiebie męże ku bitwie áby ƺli przećiw Mádiánitom / y wykonáli pomſtę PAnſką nád nimi.
Po tyśiącu z káżdego pokolenia / ze wƺyſtkich pokoleń Izráelſkich / wyślećie ná wojnę.
Y wypráwili z tyśiąców Izráelſkich / po tyśiącu <i>z káżdego</i> pokolenia / dwánaśćie tyśięcy ludźi gotowych do bitwy.
Y wyſłał je Mojzeƺ / po tyśiącu <i>z káżdego</i> pokolenia / ná wojnę : <i>poſłał</i> też z nimi Fineeſá / Syná Eleázárá Kápłaná ná wojnę / á naczynia święte / y trąby do trąbienia <i>były</i> w ręku jego.
Tedy zwiedli bitwę z Mádiánitámi / jáko rozkazał PAN Mojzeƺowi : y pobili wƺyſtkie mężczyzny.
Króle też Mádiáńſkie pobili / miedzy inƺymi pobitymi ich / Ewiego / y Rechemá / y Surá / y Hurá / y Rebáhá / piąći Królów Mádiáńſkich : y Báláámá Syná Beorowego / zábili mieczem.
Y pobráli w niewolą Synowie Izráelſcy żony Mádiáńczyków / y dźiatki ich / y wƺyſtko bydło ich / y wƺyſtkie trzody ich / y wƺyſtkie májętnośći ich pobráli :
A wƺyſtkie miáſtá ich / w których mieƺkáli / y wƺyſtkie zamki ich / popalili ogniem :
Y pobráli wƺyſtkie łupy / y wƺyſtkie plony / z ludzi / y z bydłá /
Y przywiedli do Mojzeƺá / y do Eleázárá Kápłaná / y do zgromádzenia Synów Izráelſkich / więźnie / y łupy / y korzyśći / do Obozu / ná polá Moábſkie / które ſą nád Jordanem / <i>przećiw</i> Jerichu.
Y wyƺli / Mojzeƺ y Eleázár Kápłan / y wƺyſtkie Kśiążętá zgromádzenia / przećiwko nim / przed Obóz.
Y rozgniewał śię bárzo Mojzeƺ / ná Hetmány wojſká onego / ná Pułkowniki / y ná Rotmiſtrze / którzy śię byli wróćili z oney bitwy.
Y mówił do nich Mojzeƺ / Przeczżeśćie żywo záchowáli wƺyſtkie niewiáſty?
Gdyż te ſą które Synom Izráelſkim / zá rádą Báláámową / dáły przyczynę do przeſtępſtwá / przećiw PAnu / przy báłwánie Fegor : z kąd byłá przyƺłá plagá / ná zgromádzenie PAnſkie.
Przetoż teraz / pozábijajćie wƺyſtkie mężczyzny z dźieći / y káżdą niewiáſtę / która poznáłá mężá / obcując z nim / zábijćie :
Ale wƺyſtkie dźieweczki z białych głów / które nie poznáły łożá męſkiego / żywo záchowajćie ſobie :
A wy <i>ſámi</i> zoſtańćie w Namiećiech zá Obozem / przez śiedm dni : káżdy który kogo zábił / y który śię dotykał zábitego / oczyśćićie śię dniá trzećiego : á dniá śiódmego śiebie / y więźnie wáƺe :

Y wƺelką ƺátę / y wƺelkie naczynie ſkórzáne / y wƺyſtko co urobiono z koźiey śierśći / y wƺelkie naczynie drzewiáne / oczyśćićie.
Y rzekł Eleázár Kápłan do żołnierſtwá / które chodźiło ná wojnę : Táć jeſt uſtáwá zakonna / którą był rozkazał PAN Mojzeƺowi :
Złoto jednák y śrebro / miedź / żelázo / Cynę y Ołów :
Y káżdą rzecz / która znieść może Ogień / wypráwićie przez ogień / á będźie oczyśćiona : wƺákże <i>pierwey</i> wodą oczyśćienia / będźie oczyśćiona : ále wƺyſtko co nie może znieść Ogniá / wodą oczyśćićie.
Upierzećie też ƺáty wáƺe dniá śiódmego / y czyſtymi będźiećie : á potym wnidźiećie do Obozu.
Zátym rzekł PAN do Mojzeƺá / mówiąc :
Zbierz ſummę korzyśći pobránych / z ludźi / y z bydłá / ty / y Eleázár Kápłan / y przedniejƺy z Ojców w ludu /
Y rozdźieliƺ te łupy ná dwie częśći / miedzy żołnierze / którzy ná wojnę wychodźili / y miedzy wƺyſtko zgromádzenie.
Odbierzeƺ też dział ná PAná / od mężów Rycerſkich / którzy byli wyƺli ná wojnę : po jednemu od piąći ſet / z ludźi y z wołów / y z Oſłów / y z Owiec :
A z połowice ich weźmiećie / y oddaćie Eleázárowi Kápłanowi / ná Ofiárę podnoƺenia PANU.
A z połowice Synów Izráelſkich / weźmieƺ jednę część z piąćidźieśiąt / z ludźi / z wołów / z Oſłów y z Owiec / y z wƺelkiego bydłá / y oddaƺ to Lewitom / trzymájącym ſtraż w Przybytku PAnſkim.
Y uczynił Mojzeƺ y Eleázár Kápłan / jáko rozkazał PAN Mojzeƺowi.
A było oney korzyśći / z pozoſtáłych łupów / które rozchwyćił lud wojenny : Owiec / ƺeść kroć ſto tyśięcy / y śiedmdźieśiąt tyśięcy / y pięć tyśięcy :
Wołów záś / śiedmdźieśiąt y dwá tyśiącá :
A Oſłów ƺeśćdźieśiąt tyśięcy y jeden.
A ludźi z białych głów / które nie poznáły obcowánia z mężem / wƺyſtkich <i>było</i> trzydźieśći y dwá tyśiącá.
Y doſtáłá śię połowicá dźiału / tym co wychodźili ná wojnę : liczbá Owiec / trzy kroć ſto tyśięcy / y trzydźieśći tyśięcy / y śiedm tyśięcy / y pięć ſet.
Doſtáło śię też dźiału ná PAná / Owiec ƺeść ſet / śiedmdźieśiąt y pięć.
A z wołów trzydźieśći y ƺeść tyśięcy : á dźiału z nich PAnu / śiedmdźieśiąt y dwá.
Oſłów też trzydźieśći tyśięcy y pięć ſet : á dźiału z nich PAnu / ƺeśćdźieśiąt y jeden.
Przytym ludu ƺesnaśćie tyśięcy : á dźiału z nich PAnu / trzydźieśći y dwoje ludźi.
Y oddał Mojzeƺ dźiał ná Ofiárę PAnu / Eleázárowi Kápłanowi / jáko był rozkazał PAN Mojzeƺowi /
A <i>z drugiey</i> połowice Synów Izráelſkich / którą wźiął Mojzeƺ od mężów / którzy byli wyƺli ná wojnę :
( A połowicá <i>należąca</i> zgromádzeniu / byłá : Owiec trzy kroć ſto tyśięcy / y trzydźieśći tyśięcy / śiedm tyśięcy y pięć ſet.
A Wołów trzydźieśći y ƺeść tyśięcy.
A Oſłów trzydźieśći tyśięcy / y pięć ſet.
A ludu ƺeſnaśćie tyśięcy ) :
Wźiął Mojzeƺ z tey połowice <i>należącey</i> Synom Izráelſkim jednę część / z piąćidźieśiąt / z ludźi / y z bydłá / y dał to Lewitom / trzymájącym ſtraż Przybytku PAnſkiego / jáko był rozkazał PAN Mojzeƺowi.
Tedy przyƺli do Mojzeƺá / Hetmáni Wojſká / Pułkownicy / y Rotmiſtrze /
Y mówili do niego : <i>my</i> ſłudzy twoji przynieśliſmyć poczet mężów wojennych / którzy <i>byli</i> pod ſpráwą náƺą / á nie zginął z nas y jeden.
A ták przynieśliſmy tu ná Ofiárę PAnu / káżdy czego nábył / naczynie złote / zapony / y mánelle / pierśćienie / y nauƺnice / y łáńcuƺki dla oczyśćienia duƺ náƺych przed PAnem.
Odebrał tedy Mojzeƺ y Eleázár Kápłan / ono złoto od nich / z wƺelákiem naczyniem <i>z niego</i> urobionym.
A było onego wƺyſtkiego złotá ofiárowánego / które ofiárowáli PAnu / ƺeſnaśćie tyśięcy / śiedm ſet y pięćdźieśiąt Syklów / od Pułkowników / y od Rotmiſtrzów.
( Bo żołnierze / co łupem doſtáli / ſobie otrzymáli. )
A wźiąwƺy Mojzeƺ y Eleázár Kápłan / ono złoto / od Pułkowników y Rotmiſtrzów / wnieśli je do Namiotu zgromádzenia / ná pámiątkę Synów Izráelſkich / przed PAnem.
